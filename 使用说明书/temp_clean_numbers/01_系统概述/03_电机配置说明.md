# 电机配置说明

## Robstride 电机配置详解

本文档详细说明 OpenArmX 系统中使用的 Robstride 电机型号、参数、ID分配和映射关系。

##  CAN总线配置

### CAN通道分配

OpenArmX 使用**双CAN总线架构**,左右臂独立控制:

```
┌───────────────┐
│ 控制计算机     │
└───┬───────┬───┘
    │       │
┌───▼───┐ ┌─▼─────┐
│ can0  │ │ can1  │
└───┬───┘ └───┬───┘
    │         │
┌───▼───────┐ ┌───▼───────┐
│ 左臂+左夹爪│ │ 右臂+右夹爪│
│ (8个电机)  │ │ (8个电机)  │
└───────────┘ └───────────┘
```

**优势:**
- ✅ 总线负载分离,降低通信冲突
- ✅ 独立故障隔离
- ✅ 更高的通信可靠性

| **2** | 左肩关节2 | RS04 | left_joint_2 | 肩部关节2 |
| **3** | 左大臂关节3 | RS03 | left_joint_3 | 大臂关节1 |

#### can1 (右臂+右夹爪)

**⚠️ 重要提示:**
- 每条CAN总线上的电机ID必须**唯一**(1-8)
- 左右臂可以使用**相同的ID**(因为在不同总线上)
- **不得**随意更改电机ID,否则会导致控制混乱

##  电机控制参数

### MIT模式参数

Robstride 电机使用 **MIT模式**进行运动控制:

**控制命令格式:**
```
位置(pos):   ±12.57 rad (所有型号)
速度(vel):   RS04: ±15 rad/s, RS03: ±20 rad/s, RS00: ±33 rad/s
扭矩(tau):   根据电机型号
KP(kp):      RS04/RS03: 0~5000, RS00: 0~500
KD(kd):      RS04/RS03: 0~100, RS00: 0~5
```

**各型号参数范围:**

| 型号 | 峰值扭矩 | 额定扭矩 | 最大速度 | KP范围 | KD范围 |
|------|---------|---------|---------|--------|--------|
| RS04 | ±120 Nm | 40 Nm | ±15 rad/s | 0-5000 | 0-100 |
| RS03 | ±60 Nm | 20 Nm | ±20 rad/s | 0-5000 | 0-100 |
| RS00 | ±14 Nm | 5 Nm | ±33 rad/s | 0-500 | 0-5 |

### 推荐KP/KD参数

#### 按应用场景推荐

| 场景 | 关节1-4 (RS04/RS03) | 关节5-7 (RS00) | 夹爪 (RS00) |
|------|-------------------|---------------|------------|
| **示教模式** | | | |
| KP | 1.0 | 0.1 | 0.1 |
| KD | 0.1 | 0.05 | 0.05 |
| **常规运动(默认)** | | | |
| KP | 50.0 | 10.0 | 5.0 |
| KD | 2.5 | 0.5 | 0.5 |
| **高刚性模式** | | | |
| KP | 500.0 | 50.0 | 50.0 |
| KD | 10.0 | 1.0 | 1.0 |

**⚠️ 注意:**
- 以上参数来自代码文件：`openarmx_hardware/include/openarmx_hardware/v10_simple_hardware.hpp`
- KP过大会导致**震荡和噪声**，甚至电机会有振动
- KD过大会导致**阻尼过强，响应慢**
- 不同负载需要**重新调整参数**
- **示教模式**：极低刚性，方便手动拖动示教
- **常规运动**：系统默认参数，平衡性能和稳定性
- **高刚性模式**：高精度定位，但可能产生噪声

##  零位定义

### 零位姿态

**零位定义:** 机械臂处于标准初始姿态时的关节角度。

**零位姿态描述:**
```
         Z(向上)
         │
         │
    ┌────┴────┐
    │  基座   │
    └────┬────┘
         │
    ┌────▼────┐
    │ 左臂/右臂│
    │ (垂直下垂)│
    │    │    │
    │    │    │
    │    │    │
    │    ▼    │
    │  夹爪   │
    │ (张开)  │
    └─────────┘
```

**各关节零位角度:**

| 关节 | 零位角度 | 姿态描述 |
|------|---------|---------|
| joint_1 | 0 rad | 肩部水平 |
| joint_2 | 0 rad | 肩部中立 |
| joint_3 | 0 rad | 无旋转 |
| joint_4 | 0 rad | 肘部伸直 |
| joint_5 | 0 rad | 腕部中立 |
| joint_6 | 0 rad | 腕部中立 |
| joint_7 | 0 rad | 无旋转 |
| gripper | 0 rad | 完全张开 |

### 设置零位

**方法一: 使用GUI工具**
```bash
# 启动电机管理界面
cd ~/openarmx_robotstride_ws-cc/src/motor_manage_openarmx
python3 main.py

# 在界面中:
#  点击"启动CAN"
#  点击"启动电机"
#  手动调整机械臂到零位姿态
#  点击"设置零位"
```

**方法二: 使用命令行脚本**
```bash
cd ~/openarmx_robotstride_ws-cc/src/motor_tests_openarmx_com

#  手动调整机械臂到零位姿态(±30度以内)

#  执行零位设置脚本
python3 set_motor_zeros_setMIT.py

# 脚本会:
# - 读取当前电机位置
# - 检查是否在±30度范围内
# - 设置当前位置为零位
# - 保存到电机Flash
```

**⚠️ 零位设置注意事项:**
1. 设置零位前确保机械臂处于**安全姿态**
2. 零位偏差不得超过**±30度**
3. 零位设置后需要**重启电机**生效
4. 定期检查零位准确性(推荐每月一次)

### 验证零位

### 关键状态参数

| 参数 | 正常范围 | 警告阈值 | 危险阈值 |
|------|---------|---------|---------|
| **温度** | <60°C | 60-80°C | >80°C |
| **位置误差** | <0.1 rad | 0.1-0.5 rad | >0.5 rad |
| **力矩** | 根据负载 | >80%额定 | >峰值 |
| **错误码** | 0x00 | 非零 | 通信超时 |

**温度监控建议:**
- ✅ 每次使用前检查温度
- ✅ 连续运行时每15分钟检查一次
- ⚠️ 温度>70°C时降低负载或速度
- 🔴 温度>80°C时**立即停止运动**

### 错误码说明

| 错误码 | 含义 | 处理方法 |
|-------|------|---------|
| 0x00 | 正常 | - |
| 0x01 | 过压 | 检查电源电压 |
| 0x02 | 欠压 | 检查电源供应 |
| 0x04 | 过流 | 降低负载/KP |
| 0x08 | 过温 | 停止运动,等待冷却 |
| 0x10 | 编码器故障 | 重启电机,仍故障则更换 |
| 0x20 | 通信超时 | 检查CAN连接 |

##  常见问题

### 电机ID冲突

**问题:** 同一CAN总线上有两个相同ID的电机

**现象:**
- 电机行为混乱
- 部分电机无响应
- CAN总线错误增加

**解决:**
1. 使用电机配置工具查看ID
2. 修改冲突电机的ID
3. 更新motor_mapping.yaml

### 电机无响应

**可能原因:**
1. CAN总线未启动
2. 电机未使能
3. ID配置错误
4. 线缆连接问题

**排查步骤:**
```bash
#  检查CAN接口
ip link show can0

#  检查CAN统计
ip -details -statistics link show can0

#  监听CAN总线
candump can0

#  检查电机状态
python3 check_motor_status.py
```

### 电机抖动

**可能原因:**
- KP参数过大
- KD参数过小
- 机械松动
- 负载过大

**解决方法:**
1. 降低KP值(减小20-30%)
2. 增加KD值(增加20-30%)
3. 检查机械连接
4. 降低运动速度

### 电机过热

**处理流程:**
1. 立即停止运动
2. 让电机自然冷却(不要用水)
3. 检查负载是否过大
4. 检查KP/KD是否过高
5. 降低运动速度和加速度
6. 冷却后重新测试
