##  电机异响

### 症状类型

**类型1: 尖锐啸叫**
```
原因: KP/KD过高,高频震荡
特征: 持续高频噪声
```

**类型2: 咔咔声**
```
原因: 机械卡滞,碰撞
特征: 间歇性撞击声
```

**类型3: 嗡嗡声**
```
原因: 电流噪声,PWM频率
特征: 低频嗡鸣
```

### 诊断方法

```python
#!/usr/bin/env python3
"""
电机异响诊断
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motor = Motor(bus0, 1)

motor.enable()
time.sleep(1)

# 测试1: 不同KP/KD
print("测试不同参数...")

test_params = [
    (10, 0.5, "低增益"),
    (20, 1.0, "中等增益"),
    (50, 2.5, "高增益"),
    (100, 5.0, "极高增益")
]

for kp, kd, desc in test_params:
    print(f"\n{desc}: KP={kp}, KD={kd}")
    input("按回车测试,注意听声音...")

    motor.set_position(0.5, 0, kp, kd, 0)
    time.sleep(3)

    motor.set_position(0.0, 0, kp, kd, 0)
    time.sleep(3)

motor.disable()
```

### 解决方法

**针对啸叫:**
```python
# 降低KP/KD
kp = 15  # 从50降低
kd = 0.8  # 从2.5降低

motor.set_position(target, 0, kp, kd, 0)
```

**针对咔咔声:**
```python
#  检查机械装配
#  检查是否碰撞
#  检查关节限位

# 添加限位检查
if position < joint_limit_min or position > joint_limit_max:
    print("警告: 超出关节限位")
```

**针对嗡嗡声:**
```
通常为正常现象
如果过大:
- 检查电源质量
- 检查接地
- 降低电流(降低KP)
```


##  震荡/抖动

### 症状

```
- 到达目标后持续震荡
- 静止时微小抖动
- 运动过程中震荡
```

### 原因分析

**原因1: 参数不当**
```
KP过高 → 响应过快 → 超调 → 震荡
KD不足 → 阻尼小 → 无法抑制震荡
```

**原因2: 机械问题**
```
- 装配松动
- 轴承磨损
- 负载不平衡
```

**原因3: 控制频率**
```
频率过低 → 采样不足 → 控制不稳定
```

### 解决方法

**方法1: 调整参数**
```python
# 降低KP,增加KD

# 震荡前
kp, kd = 50, 1.0

# 调整后
kp, kd = 30, 2.0

motor.set_position(target, 0, kp, kd, 0)
```

**方法2: 添加死区**
```python
def control_with_deadband(motor, target, deadband=0.01):
    """带死区的控制"""
    current = motor.get_feedback()['position']
    error = abs(target - current)

    if error < deadband:
        # 在死区内,不控制
        return

    motor.set_position(target, 0, kp=20, kd=1.0, torque=0)

# 使用
control_with_deadband(motor, 0.5, deadband=0.02)
```

**方法3: 滤波**
```python
class LowPassFilter:
    def __init__(self, alpha=0.1):
        self.alpha = alpha
        self.filtered_value = 0.0

    def filter(self, new_value):
        self.filtered_value = self.alpha * new_value + \
                             (1 - self.alpha) * self.filtered_value
        return self.filtered_value

# 使用

while True:
    fb = motor.get_feedback()
    filtered_pos = lpf.filter(fb['position'])

    # 使用滤波后的位置进行控制
```


##  过热保护

### 症状

```
- 电机停止响应
- 错误码: 0x08
- 温度传感器读数>80°C
```

### 温度监控

```python
#!/usr/bin/env python3
"""
温度监控
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motors = [Motor(bus0, i) for i in range(1, 8)]

for m in motors:
    m.enable()

print("温度监控 (Ctrl+C退出)")

try:
    while True:
        print(f"\n[{time.strftime('%H:%M:%S')}]")

        for i, motor in enumerate(motors):
            fb = motor.get_feedback()

            # 温度 (如果支持)
            temp = fb.get('temperature', 'N/A')

            # 力矩 (间接反映发热)
            torque = abs(fb['torque'])

            status = "🟢"
            if isinstance(temp, (int, float)):
                if temp > 70:
                    status = "🔴"
                elif temp > 60:
                    status = "🟡"

            print(f"  {status} Motor {i+1}: 温度={temp}°C, 力矩={torque:.2f}Nm")

        time.sleep(5)

except KeyboardInterrupt:
    for m in motors:
        m.disable()
```

### 降温措施

**立即措施:**
```
1. 停止电机运动
2. 失能电机
3. 等待自然冷却(10-30分钟)
```

**预防措施:**
```python
# 降低负载

# 降低运动速度

```

**环境改善:**

##  位置偏差

### 症状

```
- 实际位置与目标位置差异大
- 无法到达目标
- 位置漂移
```

### 诊断

```python
#!/usr/bin/env python3
"""
位置精度测试
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motor = Motor(bus0, 1)

motor.enable()
time.sleep(1)

# 测试多个目标位置
targets = [0.0, 0.5, 1.0, 0.5, 0.0, -0.5, -1.0, 0.0]

print("目标位置 | 实际位置 | 误差")
print("-" * 40)

for target in targets:
    motor.set_position(target, 0, kp=30, kd=1.5, torque=0)
    time.sleep(3)

    fb = motor.get_feedback()
    actual = fb['position']
    error = abs(target - actual)

    print(f"{target:8.3f} | {actual:8.3f} | {error:6.4f}")

motor.disable()
```

### 原因与解决

**原因1: 零位未校准**
```bash
# 重新校准零位
python3 set_motor_zeros_setMIT.py

# 断电重启
# 验证
python3 control_motor_gohome.py
```

**原因2: KP不足**
```python
# 增加KP
kp = 50  # 从20提高
kd = 2.5  # 相应增加KD

motor.set_position(target, 0, kp, kd, 0)
```

**原因3: 负载过大**

##  失控运动

### 症状

```
⚠️ 危险! 立即按急停!

- 电机高速失控旋转
- 运动超出预期
- 无法停止
```

### 紧急处理

```
1. 立即按急停按钮
2. 断开48V电源
3. 不要靠近机械臂
4. 等待完全停止
```

### 原因分析

**原因1: 零位错误**
```
零位设置错误,导致目标位置计算错误
→ 电机认为需要大幅度运动
```

**原因2: 参数错误**
```
KP/KD设置错误
→ 正反馈 → 失控
```

**原因3: 通信错误**
```
CAN消息错误,目标位置错误
→ 电机接收到异常目标
```

### 预防措施

**软件限位:**
```python
class SafeMotor:
    def __init__(self, motor, min_pos, max_pos):
        self.motor = motor
        self.min_pos = min_pos
        self.max_pos = max_pos

    def safe_set_position(self, target, vel, kp, kd, torque):
        """带限位检查的控制"""
        # 限位检查
        if target < self.min_pos or target > self.max_pos:
            print(f"⚠️ 目标{target}超出限位[{self.min_pos}, {self.max_pos}]")
            return False

        # 当前位置检查
        current = self.motor.get_feedback()['position']
        if current < self.min_pos or current > self.max_pos:
            print(f"⚠️ 当前位置{current}超出限位,紧急停止")
            self.motor.disable()
            return False

        # 安全控制
        self.motor.set_position(target, vel, kp, kd, torque)
        return True

# 使用
safe_motor = SafeMotor(motor, min_pos=-3.0, max_pos=3.0)
safe_motor.safe_set_position(0.5, 0, 20, 1.0, 0)
```

**速度监控:**
```python
def velocity_monitor(motor, max_velocity=5.0):
    """速度监控"""
    fb = motor.get_feedback()
    vel = abs(fb['velocity'])

    if vel > max_velocity:
        print(f"⚠️ 速度过高: {vel:.2f} rad/s,紧急停止")

```


##  错误码处理

### 错误码表

| 错误码 | 含义 | 处理 |
|--------|------|------|
| 0x00 | 正常 | 无 |
| 0x01 | 过压 | 检查电源,降低电压 |
| 0x02 | 欠压 | 检查电源,提高电压 |
| 0x04 | 过流 | 降低负载,检查短路 |
| 0x08 | 过温 | 停止运动,冷却 |
| 0x10 | 编码器故障 | 检查/更换电机 |
| 0x20 | 通信超时 | 检查CAN连接 |

### 错误码读取

```python
#!/usr/bin/env python3
"""
读取电机错误码
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')

ERROR_MESSAGES = {
    0x00: "正常",
    0x01: "过压",
    0x02: "欠压",
    0x04: "过流",
    0x08: "过温",
    0x10: "编码器故障",
    0x20: "通信超时"
}

for motor_id in range(1, 9):
    motor = Motor(bus0, motor_id)

    try:
        fb = motor.get_feedback()

        # 错误码 (如果SDK支持)
        error_code = fb.get('error_code', 0x00)

        if error_code != 0x00:
            error_msg = ERROR_MESSAGES.get(error_code, f"未知错误({error_code:#04x})")
            print(f"Motor {motor_id}: ✗ {error_msg}")
        else:
            print(f"Motor {motor_id}: ✓ 正常")

    except Exception as e:
        print(f"Motor {motor_id}: ✗ 通信失败 ({e})")

    time.sleep(0.1)
```

### 错误恢复

```python
def recover_from_error(motor):
    """错误恢复流程"""
    print("尝试恢复电机...")

    # 1. 失能
    motor.disable()
    time.sleep(1)

    # 2. 重新使能
    motor.enable()
    time.sleep(1)

    # 3. 检查状态
    try:
        fb = motor.get_feedback()
        error_code = fb.get('error_code', 0x00)

        if error_code == 0x00:
            print("✓ 恢复成功")
            return True
        else:
            print(f"✗ 仍有错误: {error_code:#04x}")
            return False

    except:
        print("✗ 恢复失败")
        return False
```


##  诊断流程图

```
电机异常
    ↓
检查供电 (48V)
    ↓
检查CAN通信 (candump)
    ↓
检查电机状态 (check_motor_status.py)
    ↓
        ├─ 无响应 → 检查连接,重启电机
        ├─ 异响 → 调整KP/KD,检查机械
        ├─ 震荡 → 降KP,增KD
        ├─ 过热 → 停止,冷却,降负载
        ├─ 位置偏差 → 校准零位,增KP
        ├─ 失控 → 急停,检查零位
        └─ 错误码 → 查表,对应处理
```
