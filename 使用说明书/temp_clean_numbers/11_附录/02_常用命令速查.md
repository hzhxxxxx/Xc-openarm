# 常用命令速查

## OpenArmX常用命令快速参考

一页纸速查表,快速查找常用命令。


##  系统启动

### 基础启动

```bash
# CAN总线管理
python3 en_all_can.py              # 启动CAN
python3 dis_all_can.py             # 关闭CAN

# 电机管理
python3 en_all_motors.py           # 使能所有电机
python3 dis_all_motors.py          # 失能所有电机
python3 check_motor_status.py     # 检查电机状态

# 回零
python3 control_motor_gohome.py    # 回零操作

# 启动系统
ros2 launch openarmx_bringup openarm.launch.py

# 启动MoveIt
ros2 launch openarm_moveit_config demo.launch.py
```

### 分步启动

```bash
#  硬件接口
ros2 launch openarmx_bringup hardware_only.launch.py

#  控制器
ros2 control load_controller joint_state_broadcaster
ros2 control set_controller_state joint_state_broadcaster start

ros2 control load_controller left_arm_controller
ros2 control set_controller_state left_arm_controller start

#  可视化
ros2 launch openarmx_bringup rviz.launch.py
```


##  CAN总线命令

### CAN接口管理

```bash
# 查看状态
ip link show can0
ip link show can1

# 启动CAN (1 Mbps)
sudo ip link set can0 type can bitrate 1000000
sudo ip link set can0 up

# 关闭CAN
sudo ip link set can0 down

# 查看统计
ip -s link show can0
ip -s -d link show can0

# 重置错误计数
sudo ip link set can0 down && sudo ip link set can0 up
```

### CAN通信测试

```bash
# 监听消息
candump can0
candump can0 -t A              # 带时间戳
candump can0 -e                # 只看错误帧
candump can0,001:7FF           # 过滤ID

# 发送消息
cansend can0 001#0102030405060708

# 生成测试流量
cangen can0 -g 10 -I 1

# 保存日志
candump -l can0

# 查看模块
lsmod | grep can
```


##  ROS2命令

### 节点管理

```bash
# 列出节点
ros2 node list

# 节点信息
ros2 node info /controller_manager

# 节点日志
ros2 topic echo /rosout
ros2 topic echo /rosout | grep ERROR
```

### 话题操作

```bash
# 列出话题
ros2 topic list
ros2 topic list -t              # 带类型

# 话题信息
ros2 topic info /joint_states

# 查看内容
ros2 topic echo /joint_states
ros2 topic echo /joint_states --once

# 性能监控
ros2 topic hz /joint_states     # 频率
ros2 topic bw /joint_states     # 带宽
ros2 topic delay /joint_states  # 延迟

# 发布消息
ros2 topic pub /test std_msgs/msg/String "data: 'test'"
ros2 topic pub -1 /test std_msgs/msg/String "data: 'once'"  # 发送一次
```

### 服务调用

```bash
# 列出服务
ros2 service list
ros2 service list -t            # 带类型

# 服务类型
ros2 service type /controller_manager/list_controllers

# 调用服务
ros2 service call /controller_manager/list_controllers \
    controller_manager_msgs/srv/ListControllers

# 查看服务定义
ros2 interface show controller_manager_msgs/srv/ListControllers
```

### 参数管理

```bash
# 列出参数
ros2 param list
ros2 param list /controller_manager

# 获取参数
ros2 param get /controller_manager update_rate

# 设置参数
ros2 param set /controller_manager update_rate 200

# 导出参数
ros2 param dump /controller_manager > params.yaml

# 加载参数
ros2 param load /controller_manager params.yaml
```

### Action操作

```bash
# 列出Action
ros2 action list
ros2 action list -t             # 带类型

# Action信息
ros2 action info /left_arm_controller/follow_joint_trajectory

# 发送Action Goal
ros2 action send_goal \
    /left_arm_controller/follow_joint_trajectory \
    control_msgs/action/FollowJointTrajectory \
    "{trajectory: {joint_names: ['left_joint_1']}}"
```


##  ros2_control命令

### 控制器管理

```bash
# 列出控制器
ros2 control list_controllers

# 列出硬件接口
ros2 control list_hardware_interfaces

# 列出硬件组件
ros2 control list_hardware_components

# 加载控制器
ros2 control load_controller <controller_name>

# 配置控制器
ros2 control set_controller_state <controller_name> configure

# 启动控制器
ros2 control set_controller_state <controller_name> start

# 停止控制器
ros2 control set_controller_state <controller_name> stop

# 卸载控制器
ros2 control unload_controller <controller_name>
```

### 控制器切换

```bash
# 切换控制器
ros2 control switch_controllers \
    --start-controllers left_arm_controller \
    --stop-controllers position_controller

# 强制切换
ros2 control switch_controllers \
    --start-controllers left_arm_controller \
    --stop-controllers position_controller \
    --strictness 2
```


##  编译与构建

### colcon命令

```bash
# 完整编译
cd ~/openarmx_robotstride_ws-cc
colcon build

# 编译特定包
colcon build --packages-select openarmx_hardware

# 编译包及其依赖
colcon build --packages-up-to openarmx_hardware

# 符号链接安装(开发模式)
colcon build --symlink-install

# 并行编译
colcon build --parallel-workers 4

# 详细输出
colcon build --event-handlers console_direct+

# 清理
rm -rf build/ install/ log/

# 测试
colcon test
colcon test --packages-select openarmx_hardware
colcon test-result --verbose
```

### 依赖管理

```bash

rosdep install --from-paths src --ignore-src -r -y

# 更新rosdep
rosdep update

# 检查依赖
rosdep check --from-paths src --ignore-src
```

### Python快速测试

```python
import can
from robstride import Motor

# 连接
bus = can.interface.Bus(channel='can0', bustype='socketcan')
motor = Motor(bus, motor_id=1)

# 使能
motor.enable()

# 位置控制
motor.set_position(position=0.5, kp=30.0, kd=1.0)

# 获取反馈
fb = motor.get_feedback()
print(f"Position: {fb['position']:.3f} rad")

# 失能
motor.disable()
```

##  快速诊断脚本

### 系统检查

```bash
#!/bin/bash
# 快速系统检查

echo "===== OpenArmX系统检查 ====="

# CAN
echo -n "CAN0: "
ip link show can0 | grep -q "UP,LOWER_UP" && echo "✓" || echo "✗"

# 节点
echo -n "Controller Manager: "
ros2 node list | grep -q controller_manager && echo "✓" || echo "✗"

# 控制器
echo "控制器状态:"
ros2 control list_controllers

# 关节状态
echo -n "Joint States: "
timeout 2 ros2 topic echo /joint_states -n 1 &>/dev/null && echo "✓" || echo "✗"

echo "===== 检查完成 ====="
```

### 性能检查

```bash
#!/bin/bash
# 性能检查

echo "===== 性能检查 ====="

# 话题频率
echo "Joint States频率:"
timeout 3 ros2 topic hz /joint_states 2>&1 | grep "average rate"

# CPU使用
echo -e "\nCPU使用率:"
top -b -n 1 -p $(pgrep controller_manager) | tail -1 | awk '{print $9"%"}'

# 内存使用
echo -e "\n内存使用:"
free -h | grep "Mem:" | awk '{print "Used: " $3 " / " $2}'

# CAN统计
echo -e "\nCAN错误:"
ip -s link show can0 | grep -A 4 "RX errors"

echo "===== 检查完成 ====="
```

##  故障排查速查

| 问题 | 检查命令 | 解决方案 |
|------|----------|----------|
| CAN无响应 | `ip link show can0` | `python3 en_all_can.py` |
| 电机无反馈 | `candump can0` | 检查电源,重启CAN |
| 控制器未启动 | `ros2 control list_controllers` | 重新加载控制器 |
| 关节状态无数据 | `ros2 topic hz /joint_states` | 重启硬件接口 |
| CPU使用率高 | `htop` | 降低update_rate |
| 规划失败 | 查看MoveIt日志 | 调整碰撞检测 |
