# 控制器管理

## ros2_control控制器高级管理

掌握控制器的加载、卸载、切换和监控技巧。


##  控制器管理概述

### Controller Manager

**作用:**
```
ros2_control的核心组件
管理所有控制器
协调硬件接口
提供ROS2服务接口
```

**主要功能:**
```
□ 加载/卸载控制器
□ 配置/激活/停止控制器
□ 监控控制器状态
□ 切换控制器
□ 硬件接口管理
```

### 控制器生命周期

```
                  ┌─────────────┐
                  │ unconfigured│
                  └──────┬──────┘
                         │ configure
                         ↓
                  ┌──────────┐
             ┌────┤ inactive ├────┐
             │    └────┬─────┘    │
             │         │          │
       cleanup│    activate       │
             │         ↓          │
             │    ┌────────┐     │
             └───→│ active │     │
                  └────┬───┘     │
                       │         │
                  deactivate     │
                       └─────────┘

unconfigured: 未配置
inactive: 已配置,未激活
active: 激活运行
```

**状态转换:**
```
unconfigured → configure → inactive
inactive → activate → active
active → deactivate → inactive
inactive → cleanup → unconfigured
```


##  控制器命令行管理

### 列出控制器

```bash
# 列出所有控制器
ros2 control list_controllers

# 输出示例:
# joint_state_broadcaster[joint_state_broadcaster/JointStateBroadcaster] active
# left_arm_controller[joint_trajectory_controller/JointTrajectoryController] active
# right_arm_controller[joint_trajectory_controller/JointTrajectoryController] active
```

**输出格式:**
```
控制器名称[类型] 状态

状态:
  active: 激活
  inactive: 未激活
  unconfigured: 未配置
```

### 查看硬件接口

```bash
# 列出硬件接口
ros2 control list_hardware_interfaces

# 输出示例:
# command interfaces:
#   left_joint_1/position [available] [claimed]
#   left_joint_2/position [available] [claimed]
#   ...
# state interfaces:
#   left_joint_1/position [available]
#   left_joint_1/velocity [available]
#   ...
```

**接口状态:**
```
available: 硬件提供
claimed: 被控制器占用
unclaimed: 未被占用
```

### 加载控制器

```bash
# 加载控制器
ros2 control load_controller left_arm_controller

# 加载并启动
ros2 control load_controller --set-state start left_arm_controller

# 加载并配置(不启动)
ros2 control load_controller --set-state configure left_arm_controller
```

**参数:**
```
--set-state start: 加载后激活
--set-state configure: 加载后配置(inactive)
(无参数): 仅加载(unconfigured)
```

### 卸载控制器

```bash
# 卸载控制器
ros2 control unload_controller left_arm_controller

# 注意: 需要先停止控制器
```

**完整流程:**
```bash
#  停止
ros2 control set_controller_state left_arm_controller stop

#  卸载
ros2 control unload_controller left_arm_controller
```

### 切换控制器状态

```bash
# 配置
ros2 control set_controller_state left_arm_controller configure

# 启动
ros2 control set_controller_state left_arm_controller start

# 停止
ros2 control set_controller_state left_arm_controller stop
```

**状态转换表:**
```
当前状态        命令              目标状态
unconfigured → configure    → inactive
inactive     → start        → active
active       → stop         → inactive
inactive     → cleanup      → unconfigured (需先unload)
```

### 切换控制器

```bash
# 切换控制器 (原子操作)
ros2 control switch_controllers \
  --start-controllers left_arm_trajectory_controller \
  --stop-controllers left_arm_position_controller \
  --strictness 2 \
  --start-asap false
```

**参数说明:**
```
--start-controllers: 要启动的控制器(列表)
--stop-controllers: 要停止的控制器(列表)
--strictness: 严格程度
  1 = BEST_EFFORT: 尽力而为
  2 = STRICT: 严格,全部成功或全部失败
--start-asap: 是否立即启动
  true: 立即
  false: 等待当前循环结束
```


##  Python管理接口

### 基本管理类

```python
#!/usr/bin/env python3
"""
控制器管理Python接口
"""
import rclpy
from rclpy.node import Node
from controller_manager_msgs.srv import (
    ListControllers,
    LoadController,
    UnloadController,
    SwitchController,
    ConfigureController
)

class ControllerManager(Node):
    def __init__(self):
        super().__init__('controller_manager_client')

        # 创建服务客户端
        self.list_client = self.create_client(
            ListControllers,
            '/controller_manager/list_controllers'
        )

        self.load_client = self.create_client(
            LoadController,
            '/controller_manager/load_controller'
        )

        self.unload_client = self.create_client(
            UnloadController,
            '/controller_manager/unload_controller'
        )

        self.switch_client = self.create_client(
            SwitchController,
            '/controller_manager/switch_controller'
        )

        self.configure_client = self.create_client(
            ConfigureController,
            '/controller_manager/configure_controller'
        )

        # 等待服务可用
        self.wait_for_services()

    def wait_for_services(self):
        """等待所有服务可用"""
        services = [
            self.list_client,
            self.load_client,
            self.unload_client,
            self.switch_client,
            self.configure_client
        ]

        for service in services:
            service.wait_for_service(timeout_sec=5.0)

        self.get_logger().info("所有服务已就绪")

    def list_controllers(self):
        """列出所有控制器"""
        request = ListControllers.Request()
        future = self.list_client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        result = future.result()
        self.get_logger().info(f"找到 {len(result.controller)} 个控制器")

        for ctrl in result.controller:
            self.get_logger().info(
                f"  {ctrl.name} [{ctrl.type}] - {ctrl.state}"
            )

        return result.controller

    def load_controller(self, controller_name):
        """加载控制器"""
        request = LoadController.Request()
        request.name = controller_name

        future = self.load_client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        result = future.result()
        if result.ok:
            self.get_logger().info(f"✓ 加载成功: {controller_name}")
            return True
        else:
            self.get_logger().error(f"✗ 加载失败: {controller_name}")
            return False

    def unload_controller(self, controller_name):
        """卸载控制器"""
        request = UnloadController.Request()
        request.name = controller_name

        future = self.unload_client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        result = future.result()
        if result.ok:
            self.get_logger().info(f"✓ 卸载成功: {controller_name}")
            return True
        else:
            self.get_logger().error(f"✗ 卸载失败: {controller_name}")
            return False

    def configure_controller(self, controller_name):
        """配置控制器"""
        request = ConfigureController.Request()
        request.name = controller_name

        future = self.configure_client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        result = future.result()
        if result.ok:
            self.get_logger().info(f"✓ 配置成功: {controller_name}")
            return True
        else:
            self.get_logger().error(f"✗ 配置失败: {controller_name}")
            return False

    def switch_controllers(self, start_controllers, stop_controllers):
        """切换控制器"""
        request = SwitchController.Request()
        request.start_controllers = start_controllers
        request.stop_controllers = stop_controllers
        request.strictness = 2  # STRICT
        request.start_asap = False
        request.timeout = rclpy.duration.Duration(seconds=0.0).to_msg()

        future = self.switch_client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        result = future.result()
        if result.ok:
            self.get_logger().info(f"✓ 切换成功")
            return True
        else:
            self.get_logger().error(f"✗ 切换失败")
            return False

def main():
    rclpy.init()

    manager = ControllerManager()

    # 示例: 列出控制器
    controllers = manager.list_controllers()

    # 示例: 切换控制器
    manager.switch_controllers(
        start_controllers=['left_arm_controller'],
        stop_controllers=[]
    )

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 控制器状态监控

```python
#!/usr/bin/env python3
"""
实时监控控制器状态
"""
import rclpy
from rclpy.node import Node
from controller_manager_msgs.msg import ControllerState
from controller_manager_msgs.srv import ListControllers
import time

class ControllerMonitor(Node):
    def __init__(self):
        super().__init__('controller_monitor')

        self.list_client = self.create_client(
            ListControllers,
            '/controller_manager/list_controllers'
        )

        # 定期检查
        self.timer = self.create_timer(2.0, self.check_controllers)

    def check_controllers(self):
        """定期检查控制器状态"""
        if not self.list_client.wait_for_service(timeout_sec=1.0):
            self.get_logger().warn("Controller Manager服务不可用")
            return

        request = ListControllers.Request()
        future = self.list_client.call_async(request)

        rclpy.spin_until_future_complete(self, future, timeout_sec=1.0)

        if future.result() is None:
            return

        result = future.result()

        print("\n========== 控制器状态 ==========")
        print(f"时间: {time.strftime('%H:%M:%S')}")
        print(f"总数: {len(result.controller)}")
        print("-" * 50)

        for ctrl in result.controller:
            status_icon = "🟢" if ctrl.state == "active" else "🔴"
            print(f"{status_icon} {ctrl.name:<30} {ctrl.state}")

        print("=" * 50)

def main():
    rclpy.init()

    monitor = ControllerMonitor()

    try:
        rclpy.spin(monitor)
    except KeyboardInterrupt:
        pass

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```


##  控制器切换策略

### 零宕机切换

**场景:** 不中断运动,平滑切换控制器

```python
def smooth_switch(manager, from_controller, to_controller):
    """
    平滑切换控制器
    """
    # 1. 加载目标控制器(如果未加载)
    manager.load_controller(to_controller)

    # 2. 配置目标控制器
    manager.configure_controller(to_controller)

    # 3. 原子切换
    success = manager.switch_controllers(
        start_controllers=[to_controller],
        stop_controllers=[from_controller]
    )

    if success:
        # 4. 卸载旧控制器(可选)
        # manager.unload_controller(from_controller)
        print(f"切换成功: {from_controller} → {to_controller}")
    else:
        print("切换失败")

    return success
```

### 停止-启动切换

**场景:** 完全停止后再启动,确保安全

```python
def safe_switch(manager, from_controller, to_controller):
    """
    安全切换(先停止再启动)
    """
    import time

    # 1. 停止当前控制器
    manager.switch_controllers(
        start_controllers=[],
        stop_controllers=[from_controller]
    )

    # 2. 等待停稳
    time.sleep(1.0)

    # 3. 启动新控制器
    manager.switch_controllers(
        start_controllers=[to_controller],
        stop_controllers=[]
    )

    print(f"安全切换完成: {from_controller} → {to_controller}")
```

### 备用控制器

**场景:** 主控制器失败时自动切换到备用

```python
class FallbackController(Node):
    def __init__(self):
        super().__init__('fallback_controller')

        self.manager = ControllerManager()

        self.primary = 'left_arm_controller'
        self.fallback = 'left_arm_position_controller'

        # 监控主控制器
        self.timer = self.create_timer(1.0, self.monitor)

    def monitor(self):
        """监控主控制器,失败时切换"""
        controllers = self.manager.list_controllers()

        primary_state = None
        for ctrl in controllers:
            if ctrl.name == self.primary:
                primary_state = ctrl.state
                break

        if primary_state != 'active':
            self.get_logger().warn(f"主控制器异常: {primary_state}, 切换到备用")

            self.manager.switch_controllers(
                start_controllers=[self.fallback],
                stop_controllers=[self.primary]
            )
```

    "joint_state_broadcaster"

### Python批量管理

```python
#!/usr/bin/env python3
"""
批量控制器管理
"""
import rclpy
from controller_manager import ControllerManager

def start_all_controllers(manager, controller_list):
    """批量启动控制器"""
    failed = []

    for ctrl_name in controller_list:
        print(f"启动 {ctrl_name} ...")

        # 加载并启动
        if manager.load_controller(ctrl_name):
            if manager.configure_controller(ctrl_name):
                success = manager.switch_controllers(
                    start_controllers=[ctrl_name],
                    stop_controllers=[]
                )

                if not success:
                    failed.append(ctrl_name)
            else:
                failed.append(ctrl_name)
        else:
            failed.append(ctrl_name)

    if len(failed) == 0:
        print("✓ 所有控制器启动成功")
        return True
    else:
        print(f"✗ 以下控制器启动失败: {failed}")
        return False

def main():
    rclpy.init()

    manager = ControllerManager()

    controllers = [
        'joint_state_broadcaster',
        'left_arm_controller',
        'right_arm_controller',
        'left_gripper_controller',
        'right_gripper_controller'
    ]

    start_all_controllers(manager, controllers)

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```


##  控制器诊断

### 诊断检查

```python
#!/usr/bin/env python3
"""
控制器健康检查
"""
import rclpy
from controller_manager import ControllerManager

def diagnose_controllers(manager):
    """诊断控制器问题"""
    controllers = manager.list_controllers()

    print("\n========== 控制器诊断 ==========\n")

    issues = []

    # 检查1: 是否有控制器
    if len(controllers) == 0:
        issues.append("错误: 没有找到任何控制器")

    # 检查2: joint_state_broadcaster必须active
    jsc_found = False
    for ctrl in controllers:
        if 'joint_state_broadcaster' in ctrl.name:
            jsc_found = True
            if ctrl.state != 'active':
                issues.append(f"警告: joint_state_broadcaster状态为{ctrl.state}, 应为active")

    if not jsc_found:
        issues.append("错误: 未找到joint_state_broadcaster")

    # 检查3: 检查claimed接口
    # (需要调用list_hardware_interfaces服务)

    # 检查4: 检查配置一致性
    for ctrl in controllers:
        if ctrl.state == 'unconfigured':
            issues.append(f"提示: {ctrl.name}未配置")

    # 输出结果
    if len(issues) == 0:
        print("✓ 所有检查通过,控制器正常")
    else:
        print(f"✗ 发现 {len(issues)} 个问题:")
        for issue in issues:
            print(f"  - {issue}")

    print("\n" + "=" * 50 + "\n")

    return len(issues) == 0

def main():
    rclpy.init()

    manager = ControllerManager()

    diagnose_controllers(manager)

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 控制器参数热更新

```python
#!/usr/bin/env python3
"""
运行时更新控制器参数
"""
import rclpy
from rclpy.node import Node
from rcl_interfaces.srv import SetParameters
from rcl_interfaces.msg import Parameter, ParameterValue

class ParameterUpdater(Node):
    def __init__(self):
        super().__init__('param_updater')

        self.client = self.create_client(
            SetParameters,
            '/left_arm_controller/set_parameters'
        )

    def update_constraints(self, joint_name, trajectory_tolerance, goal_tolerance):
        """更新轨迹容差"""
        request = SetParameters.Request()

        # 轨迹容差
        traj_param = Parameter()
        traj_param.name = f'constraints.{joint_name}.trajectory'
        traj_param.value.type = 3  # DOUBLE
        traj_param.value.double_value = trajectory_tolerance

        # 目标容差
        goal_param = Parameter()
        goal_param.name = f'constraints.{joint_name}.goal'
        goal_param.value.type = 3
        goal_param.value.double_value = goal_tolerance

        request.parameters = [traj_param, goal_param]

        future = self.client.call_async(request)
        rclpy.spin_until_future_complete(self, future)

        self.get_logger().info(f"更新 {joint_name} 容差")

# 使用
rclpy.init()
updater = ParameterUpdater()
updater.update_constraints('left_joint_1', trajectory_tolerance=0.2, goal_tolerance=0.05)
rclpy.shutdown()
```

### 控制器性能监控

```python
#!/usr/bin/env python3
"""
监控控制器性能
"""
import rclpy
from rclpy.node import Node
from control_msgs.msg import JointTrajectoryControllerState

class PerformanceMonitor(Node):
    def __init__(self):
        super().__init__('perf_monitor')

        self.subscription = self.create_subscription(
            JointTrajectoryControllerState,
            '/left_arm_controller/state',
            self.state_callback,
            10
        )

        self.error_history = []

    def state_callback(self, msg):
        """监控误差"""
        if len(msg.desired.positions) > 0:
            # 计算误差
            errors = [abs(d - a) for d, a in zip(
                msg.desired.positions,
                msg.actual.positions
            )]

            max_error = max(errors)
            avg_error = sum(errors) / len(errors)

            self.error_history.append(max_error)

            # 保持最近100个样本
            if len(self.error_history) > 100:
                self.error_history.pop(0)

            # 输出统计
            if len(self.error_history) % 10 == 0:
                import statistics
                print(f"最大误差: {max_error:.4f}, "
                      f"平均误差: {avg_error:.4f}, "
                      f"标准差: {statistics.stdev(self.error_history):.4f}")

def main():
    rclpy.init()
    monitor = PerformanceMonitor()
    rclpy.spin(monitor)
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```


##  常见问题

### 控制器加载失败

**症状:** `load_controller`返回False

**排查:**
```bash
#  检查配置文件
cat openarm_controllers.yaml

#  检查控制器类型是否安装
ros2 pkg list | grep controller

#  查看日志
ros2 topic echo /rosout | grep ERROR

#  检查硬件接口
ros2 control list_hardware_interfaces
```

### 控制器无法启动

**症状:** `start`命令失败

**原因:**
- 硬件接口不可用
- 接口被其他控制器占用
- 配置错误

**解决:**
```bash
#  检查硬件接口是否claimed
ros2 control list_hardware_interfaces

#  停止冲突的控制器
ros2 control list_controllers
ros2 control set_controller_state <冲突控制器> stop

#  重新启动
ros2 control set_controller_state <目标控制器> start
```

### 切换控制器失败

**症状:** `switch_controllers`失败

**排查:**
```python
# 检查strictness设置
# strictness=2 (STRICT) 要求全部成功
# 改为strictness=1 (BEST_EFFORT) 可能解决

manager.switch_controllers(
    start_controllers=[...],
    stop_controllers=[...],
    strictness=1  # 尝试BEST_EFFORT
)
```
