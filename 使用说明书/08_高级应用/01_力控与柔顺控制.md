## 1. 力控概述

### 1.1 什么是力控

**力控制 (Force Control):**
```
控制机器人与环境的接触力
而非仅控制位置
实现柔顺交互
```

**柔顺控制 (Compliance Control):**
```
机器人"软"的行为
受力时可以让步
类似人的手臂
```

**阻抗控制 (Impedance Control):**
```
控制位置-力之间的关系
调节"刚度"和"阻尼"
MIT模式的核心
```

## 2. 阻抗控制原理

### 2.1 阻抗方程

**基本方程:**
```
F = M × ẍ + D × ẋ + K × x

F: 外力
M: 惯性 (Mass)
D: 阻尼 (Damping) ~ KD
K: 刚度 (Stiffness) ~ KP
x: 位置偏差
```

**在MIT模式中:**
```
τ = KP × (θ_target - θ_actual) + KD × (ω_target - ω_actual) + τ_feedforward

KP: 控制刚度
KD: 控制阻尼
τ_feedforward: 额外力矩
```

### 2.2 刚度调节

**高刚度 (High Stiffness):**
```
KP: 高 (50-100)
KD: 高 (2-5)

特点:
- 硬,不易推动
- 精确定位
- 快速响应
```

**低刚度 (Low Stiffness):**
```
KP: 低 (1-10)
KD: 低 (0.1-1)

特点:
- 软,易推动
- 柔顺交互
- 安全人机协作
```

**示例:**
```python
# 高刚度 - 精确定位
motor.set_position(target, 0, kp=80, kd=3, torque=0)

# 低刚度 - 示教模式
motor.set_position(current, 0, kp=3, kd=0.3, torque=0)

# 中等刚度 - 常规控制
motor.set_position(target, 0, kp=20, kd=1, torque=0)
```

## 3. 力矩控制

### 3.1 纯力矩控制

**原理:**
```
直接指定输出力矩
不管位置
实现主动力输出
```

**实现:**
```python

motor.set_position(
```

```
- 按压操作

```

### 3.2 力矩前馈

**原理:**
```
在位置控制基础上叠加力矩
实现混合控制
```

**示例:**
```python
# 重力补偿
def calculate_gravity_torque(joint_positions, joint_index):
    """
    计算关节重力力矩
    简化模型
    """
    # 假设手臂质量m=2kg,臂长l=0.5m,重力g=9.8m/s²
    m = 2.0
    l = 0.5
    g = 9.8

    # 重力力矩 τ = m*g*l*cos(θ)
    theta = joint_positions[joint_index]
    torque = m * g * l * np.cos(theta)

    return torque

# 带重力补偿的控制
joint_positions = get_current_positions()
gravity_torque = calculate_gravity_torque(joint_positions, joint_index=2)

motor.set_position(
    position=target_position,
    velocity=0.0,
    kp=20.0,
    kd=1.0,
    torque=gravity_torque  # 补偿重力
)
```

**效果:**
```
✓ 减少位置误差
✓ 降低能耗
✓ 减少电机发热
✓ 提高负载能力
```

### 3.3 恒力控制

**目标:** 保持恒定接触力

**实现:**
```python
#!/usr/bin/env python3
"""
恒力控制示例 - 抛光应用
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motor = Motor(bus0, 4)
motor.enable()

# 目标接触力
target_force = 10.0  # N

# 转换为力矩 (简化:假设力臂0.3m)
force_arm = 0.3  # m
target_torque = target_force * force_arm  # 3.0 Nm

# 初始位置
initial_position = motor.get_feedback()['position']

# 力控循环
try:
    while True:
        # 读取当前力矩
        fb = motor.get_feedback()
        current_torque = fb['torque']
        current_position = fb['position']

        # 力矩误差
        torque_error = target_torque - current_torque

        # 位置调整 (积分控制)
        position_adjustment = torque_error * 0.01  # 积分增益

        # 新目标位置
        new_target = current_position + position_adjustment

        # 发送控制命令 (低刚度 + 力矩前馈)
        motor.set_position(
            position=new_target,
            velocity=0.0,
            kp=5.0,              # 低刚度
            kd=0.5,
            torque=target_torque  # 力矩前馈
        )

        print(f"力矩: {current_torque:.2f}/{target_torque:.2f} Nm, "
              f"位置: {current_position:.3f} rad")

        time.sleep(0.01)  # 100Hz控制

except KeyboardInterrupt:
    motor.disable()
```

## 4. 柔顺控制应用

### 4.1 柔顺装配

**场景:** 插销孔装配,允许误差

```python
#!/usr/bin/env python3
"""
柔顺装配 - Peg-in-Hole
"""
import can
from robstride import Motor
import time
import numpy as np

class CompliantInsertion:
    def __init__(self, motors):
        self.motors = motors

    def insertion(self, approach_position, insert_depth):
        """
        柔顺插入
        approach_position: 接近位置
        insert_depth: 插入深度
        """
        print("[1/4] 快速接近...")
        # 阶段1: 快速移动到接近位置 (高刚度)
        for motor in self.motors:
            motor.set_position(approach_position, 0, kp=30, kd=1.5, torque=0)

        time.sleep(2)

        print("[2/4] 接触搜索...")
        # 阶段2: 慢速接触 (中等刚度,监控力矩)
        contact_detected = False
        search_step = 0.01  # 1cm步长

        for i in range(20):  # 最多搜索20cm
            current_pos = self.motors[0].get_feedback()['position']
            new_pos = current_pos - search_step

            for motor in self.motors:
                motor.set_position(new_pos, 0, kp=15, kd=1.0, torque=0)

            time.sleep(0.2)

            # 检测接触 (力矩突增)
            fb = self.motors[0].get_feedback()
            if abs(fb['torque']) > 3.0:  # 检测到阻力
                contact_detected = True
                print(f"  检测到接触, 力矩: {fb['torque']:.2f} Nm")
                break

        if not contact_detected:
            print("  未检测到接触")
            return False

        print("[3/4] 柔顺插入...")
        # 阶段3: 柔顺插入 (低刚度 + 力控)
        insert_force = 2.0  # 插入力 (Nm)
        insert_target = current_pos - insert_depth

        for i in range(100):  # 10秒插入
            fb = self.motors[0].get_feedback()
            current_pos = fb['position']

            # 如果已到达深度
            if current_pos <= insert_target:
                print("  到达目标深度")
                break

            # 柔顺控制
            for motor in self.motors:
                motor.set_position(
                    position=current_pos - 0.005,  # 缓慢推进
                    velocity=0.0,
                    kp=5.0,       # 低刚度,允许让步
                    kd=0.5,
                    torque=insert_force  # 恒力推入
                )

            time.sleep(0.1)

            # 监控力矩,过大则停止
            if abs(fb['torque']) > 10.0:
                print(f"  力矩过大 ({fb['torque']:.2f} Nm), 停止插入")
                return False

        print("[4/4] 插入完成")
        return True

# 使用
bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motors = [Motor(bus0, i) for i in range(4, 7)]  # 肘腕关节

for m in motors:
    m.enable()

inserter = CompliantInsertion(motors)
success = inserter.insertion(approach_position=-0.5, insert_depth=0.1)

for m in motors:
    m.disable()
```

### 4.2 柔顺抓取

**场景:** 抓取易碎物品

```python
#!/usr/bin/env python3
"""
柔顺抓取 - 软物体
"""
import can
from robstride import Motor
import time

bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
gripper = Motor(bus0, 8)  # 夹爪
gripper.enable()

def compliant_grasp(target_force=1.0):
    """
    柔顺抓取
    target_force: 目标夹持力 (Nm)
    """
    print("开始柔顺抓取...")

    # 初始打开
    gripper.set_position(0.0, 0, kp=10, kd=0.5, torque=0)
    time.sleep(1)

    # 慢速闭合,监控力矩
    close_step = 0.05  # 每步5度
    max_close = 1.2    # 最大闭合角度

    for i in range(int(max_close / close_step)):
        target_position = i * close_step

        # 柔顺控制
        gripper.set_position(
            position=target_position,
            velocity=0.0,
            kp=8.0,          # 中低刚度
            kd=0.6,
            torque=0.0
        )

        time.sleep(0.2)

        # 检查力矩
        fb = gripper.get_feedback()
        current_torque = abs(fb['torque'])

        print(f"位置: {fb['position']:.3f}, 力矩: {current_torque:.3f} Nm")

        # 达到目标力则停止
        if current_torque >= target_force:
            print(f"达到目标夹持力: {current_torque:.3f} Nm")
            break

    # 保持夹持 (力控)
    final_position = gripper.get_feedback()['position']

    print("保持夹持...")
    for _ in range(50):  # 保持5秒
        gripper.set_position(
            position=final_position,
            velocity=0.0,
            kp=5.0,          # 低刚度
            kd=0.5,
            torque=target_force  # 恒力夹持
        )
        time.sleep(0.1)

    print("释放...")
    gripper.set_position(0.0, 0, kp=10, kd=0.5, torque=0)
    time.sleep(1)

# 执行
compliant_grasp(target_force=0.8)
gripper.disable()
```

### 4.3 碰撞检测与退让

**场景:** 运动中检测碰撞并退让

```python
#!/usr/bin/env python3
"""
碰撞检测与自动退让
"""
import can
from robstride import Motor
import time
import numpy as np

class CollisionDetection:
    def __init__(self, motor):
        self.motor = motor
        self.torque_history = []
        self.baseline_torque = 0.0

    def calibrate_baseline(self):
        """标定无负载力矩"""
        print("标定基准力矩...")
        torques = []

        for _ in range(50):
            fb = self.motor.get_feedback()
            torques.append(abs(fb['torque']))
            time.sleep(0.02)

        self.baseline_torque = np.mean(torques)
        print(f"基准力矩: {self.baseline_torque:.3f} Nm")

    def move_with_collision_detection(self, target_position, threshold=5.0):
        """
        带碰撞检测的运动
        threshold: 碰撞检测阈值 (Nm)
        """
        print(f"移动到 {target_position:.3f} rad...")

        collision_detected = False

        for i in range(100):  # 最多10秒
            # 发送控制命令
            self.motor.set_position(
                position=target_position,
                velocity=0.0,
                kp=20.0,
                kd=1.0,
                torque=0.0
            )

            time.sleep(0.1)

            # 读取力矩
            fb = self.motor.get_feedback()
            current_torque = abs(fb['torque'])

            # 碰撞检测
            torque_deviation = current_torque - self.baseline_torque

            if torque_deviation > threshold:
                collision_detected = True
                print(f"⚠️ 检测到碰撞! 力矩: {current_torque:.2f} Nm")
                break

            # 到达目标
            if abs(fb['position'] - target_position) < 0.05:
                print("✓ 到达目标")
                break

        if collision_detected:
            # 退让
            print("退让中...")
            current_pos = self.motor.get_feedback()['position']
            retreat_pos = current_pos + 0.1  # 退让10cm

            self.motor.set_position(
                position=retreat_pos,
                velocity=0.0,
                kp=15.0,
                kd=1.0,
                torque=0.0
            )

            time.sleep(1)
            print("退让完成")

            return False

        return True

# 使用
bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
motor = Motor(bus0, 1)
motor.enable()

detector = CollisionDetection(motor)
detector.calibrate_baseline()

# 尝试移动
success = detector.move_with_collision_detection(target_position=1.0, threshold=5.0)

if not success:
    print("运动中断,检测到障碍物")

motor.disable()
```

## 5. 重力补偿

### 5.1 简单重力补偿

**单关节模型:**

```python
def gravity_compensation(joint_angle, mass=2.0, length=0.5):
    """
    计算重力补偿力矩
    joint_angle: 关节角度 (rad)
    mass: 负载质量 (kg)
    length: 力臂长度 (m)
    """
    import numpy as np

    g = 9.8  # m/s²
    torque = mass * g * length * np.cos(joint_angle)

    return torque
```

### 5.2 多关节重力补偿

**完整手臂模型:**
```python
#!/usr/bin/env python3
"""
完整手臂重力补偿
"""
import numpy as np

class GravityCompensation:
    def __init__(self):
        # 手臂参数 (简化模型)
        self.link_masses = [1.0, 0.8, 0.5, 0.3, 0.2, 0.15, 0.1]  # kg
        self.link_lengths = [0.2, 0.25, 0.25, 0.2, 0.15, 0.1, 0.05]  # m
        self.link_coms = [0.1, 0.125, 0.125, 0.1, 0.075, 0.05, 0.025]  # 质心位置

        self.g = 9.8

    def compute_gravity_torques(self, joint_positions):
        """
        计算所有关节的重力力矩
        joint_positions: [θ1, θ2, ..., θ7]
        返回: [τ1, τ2, ..., τ7]
        """
        torques = np.zeros(7)

        # 简化计算:每个关节受后续所有连杆重力影响
        for i in range(7):
            # 累加当前及后续连杆的重力力矩
            for j in range(i, 7):
                # 计算该连杆对关节i的力臂
                # (简化:仅考虑关节2, 3, 4的重力效应)
                if i < 3:  # 肩部关节
                    # 重力力矩 ≈ m*g*l*cos(θ)
                    torques[i] += self.link_masses[j] * self.g * \
                                  self.link_coms[j] * np.cos(joint_positions[i])

        return torques

# 使用
comp = GravityCompensation()

# 在控制循环中
while True:
    joint_positions = get_current_joint_positions()
    gravity_torques = comp.compute_gravity_torques(joint_positions)

    # 应用补偿
    for i, motor in enumerate(motors):
        motor.set_position(
            position=target_positions[i],
            velocity=0,
            kp=20,
            kd=1,
            torque=gravity_torques[i]  # 重力补偿
        )

    time.sleep(0.01)
```

## 7. 常见问题

### 7.1 力控不稳定

**症状:** 力矩控制时抖动或震荡

**原因:**
- KP/KD设置不当
- 力矩变化过快
- 传感器噪声

**解决:**
```python
# 1. 降低刚度
kp, kd = 5.0, 0.5  # 从20/1.0降低

# 2. 力矩平滑
def smooth_torque(target_torque, current_torque, alpha=0.1):
    return alpha * target_torque + (1 - alpha) * current_torque

# 3. 添加死区
if abs(torque_error) < 0.5:
    torque_command = 0
```

### 7.2 重力补偿不准

**症状:** 手臂仍会下垂或上翘

**原因:**
- 模型不准确
- 负载变化
- 摩擦力

**解决:**
```
1. 细化模型(多连杆,质心位置)
2. 实际测量参数
3. 在线学习/自适应
4. 添加摩擦力补偿
```

### 7.3 碰撞检测误报

**症状:** 正常运动时触发碰撞检测

**原因:**
- 阈值过低
- 未标定基准
- 加速度大

**解决:**
```python
# 1. 提高阈值
threshold = 8.0  # 从5.0提高

# 2. 动态阈值(考虑加速度)
threshold = baseline + k * abs(acceleration)

# 3. 滤波
torque_filtered = lowpass_filter(torque_raw)
```
