# 02_系统依赖安装

## 系统环境和依赖库安装指南

本文档详细说明如何安装 OpenArmX 所需的操作系统、ROS2、CAN工具和其他依赖库。

---

## 1. 操作系统安装

### 1.1 Ubuntu 22.04 LTS 安装

**下载Ubuntu:**
```
官方网站: https://ubuntu.com/download/desktop
版本: Ubuntu 22.04.3 LTS (Jammy Jellyfish)
架构: 64-bit (amd64)
文件: ubuntu-22.04.3-desktop-amd64.iso
```

**创建启动U盘:**

**Windows下:**
```
1. 下载 Rufus: https://rufus.ie/
2. 插入 ≥4GB U盘
3. 打开 Rufus
4. 选择 Ubuntu ISO 文件
5. 分区方案: GPT
6. 目标系统: UEFI
7. 点击"开始"
```

**Linux下:**
```bash
# 假设U盘设备为 /dev/sdb (请确认!!)
sudo dd bs=4M if=ubuntu-22.04.3-desktop-amd64.iso of=/dev/sdb status=progress oflag=sync
```

**安装步骤:**
```
1. 插入启动U盘,重启电脑
2. 进入BIOS,设置U盘启动
3. 选择"Install Ubuntu"
4. 语言选择: 简体中文 或 English
5. 键盘布局: Chinese 或 English (US)
6. 更新和其他软件:
   ☑ 正常安装
   ☑ 安装Ubuntu时下载更新
   ☑ 为图形和Wi-Fi硬件安装第三方软件
7. 安装类型: 清除整个磁盘并安装Ubuntu (或自定义分区)
8. 设置用户名和密码
9. 等待安装完成
10. 重启,拔出U盘
```

### 1.2 系统初始化

**更新系统:**
```bash
# 更新软件源
sudo apt update

# 升级所有软件包
sudo apt upgrade -y

# 安装基础工具
sudo apt install -y \
    curl \
    wget \
    git \
    vim \
    net-tools \
    build-essential \
    cmake \
    python3-pip
```

**设置软件源(可选,加速下载):**

**使用阿里云镜像:**
```bash
# 备份原始sources.list
sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup

# 替换为阿里云镜像
sudo sed -i 's/archive.ubuntu.com/mirrors.aliyun.com/g' /etc/apt/sources.list
sudo sed -i 's/security.ubuntu.com/mirrors.aliyun.com/g' /etc/apt/sources.list

# 更新软件源
sudo apt update
```

## 2. ROS2 Humble 安装

### 2.1 设置ROS2软件源

**设置locale:**
```bash
sudo apt update && sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8
```

**添加ROS2 apt仓库:**
```bash
# 安装curl
sudo apt install -y curl

# 添加ROS2 GPG密钥
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key \
    -o /usr/share/keyrings/ros-archive-keyring.gpg

# 添加软件源
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
    http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | \
    sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# 更新软件源
sudo apt update
```

**中国用户加速(可选):**
```bash
# 使用清华镜像
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
    https://mirrors.tuna.tsinghua.edu.cn/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | \
    sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

sudo apt update
```

### 2.2 安装ROS2 Humble

**安装完整桌面版:**
```bash
# 安装ROS2 Humble桌面完整版(包含RViz, Demo等)
sudo apt install -y ros-humble-desktop-full

# 安装开发工具
sudo apt install -y \
    ros-dev-tools \
    python3-colcon-common-extensions \
    python3-rosdep
```

**初始化rosdep:**
```bash
# 初始化rosdep
sudo rosdep init
rosdep update
```

**⚠️ rosdep init失败的解决方法:**
```bash
# 如果rosdep init报错"ERROR: cannot download default sources list"
# 使用代理或手动创建文件

# 方法1: 手动创建
sudo mkdir -p /etc/ros/rosdep/sources.list.d/
sudo curl -o /etc/ros/rosdep/sources.list.d/20-default.list \
    https://mirrors.tuna.tsinghua.edu.cn/rosdistro/rosdep/sources.list.d/20-default.list

# 方法2: 修改rosdep源(国内用户)
export ROSDISTRO_INDEX_URL=https://mirrors.tuna.tsinghua.edu.cn/rosdistro/index-v4.yaml
rosdep update
```

### 2.3 配置ROS2环境

**添加到bashrc:**
```bash
# 编辑 .bashrc
echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc

# 立即生效
source ~/.bashrc
```

**验证安装:**
```bash
# 检查ROS2版本
ros2 --version

# 应该显示: ros2 cli version: 0.XX.X

# 测试ROS2节点
ros2 run demo_nodes_cpp talker
# 在另一个终端:
ros2 run demo_nodes_cpp listener

# 如果能看到消息传递,说明安装成功
```

## 5. CAN总线工具安装

### 5.1 安装SocketCAN工具

```bash
# 安装CAN工具包
sudo apt install -y \
    can-utils \
    iproute2

# can-utils 包含:
# - candump: 监听CAN总线
# - cansend: 发送CAN帧
# - cangen: 生成CAN流量
# - canconfig: 配置CAN接口(旧版)
```

**验证安装:**
```bash
# 查看can-utils版本
candump --version

# 列出CAN接口(此时可能为空)
ip link show
```

### 5.2 加载CAN内核模块

```bash
# 加载CAN模块
sudo modprobe can
sudo modprobe can_raw
sudo modprobe can_dev
sudo modprobe vcan  # 虚拟CAN,测试用

# 查看已加载模块
lsmod | grep can

# 开机自动加载
echo "can" | sudo tee -a /etc/modules
echo "can_raw" | sudo tee -a /etc/modules
echo "can_dev" | sudo tee -a /etc/modules
```

### 5.3 安装Python CAN库

```bash
# 安装python-can
pip3 install python-can

# 或使用系统包管理器
sudo apt install -y python3-can
```

## 11. 系统优化(可选)

### 11.1 实时性能优化

**调整GRUB参数:**
```bash
# 编辑GRUB配置
sudo vim /etc/default/grub

# 修改这一行(添加参数):
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash isolcpus=2,3 nohz_full=2,3"
# isolcpus=2,3: 隔离CPU核心2,3用于实时任务
# nohz_full=2,3: 禁用这些核心的定时器中断

# 更新GRUB
sudo update-grub

# 重启生效
sudo reboot
```

**设置CPU调度器:**
```bash
# 安装cpufrequtils
sudo apt install -y cpufrequtils

# 设置为性能模式
sudo cpufreq-set -g performance

# 查看当前模式
cpufreq-info
```

### 11.2 网络延迟优化

```bash
# 禁用IPv6(可选,减少延迟)
echo "net.ipv6.conf.all.disable_ipv6 = 1" | sudo tee -a /etc/sysctl.conf
echo "net.ipv6.conf.default.disable_ipv6 = 1" | sudo tee -a /etc/sysctl.conf
sudo sysctl -p
```

```bash

# 1. 操作系统

# 检查关键ROS2包
echo "检查ROS2包..."
packages=(
    "moveit"
    "ros2_control"
    "joint_trajectory_controller"
    "urdf"
    "xacro"
)

for pkg in "${packages[@]}"; do
    if ros2 pkg prefix $pkg &> /dev/null; then
        echo "✓ $pkg"
    else
        echo "✗ $pkg - 未安装!"
    fi
done

# 检查Python模块
echo "检查Python模块..."
python3 << EOF
import sys

modules = {
    'rclpy': 'ROS2 Python',
    'can': 'Python-CAN',
    'PySide6': 'PySide6',
    'numpy': 'NumPy',
    'yaml': 'PyYAML'
}

for module, name in modules.items():
    try:
        __import__(module)
        print(f"✓ {name}")
    except ImportError:
        print(f"✗ {name} - 未安装!")
EOF

echo "====================================="
```

## 13. 常见问题

### 13.1 rosdep init失败

**问题:** `ERROR: cannot download default sources list`

**解决方法:**
```bash
# 使用国内镜像
sudo mkdir -p /etc/ros/rosdep/sources.list.d/
sudo curl -o /etc/ros/rosdep/sources.list.d/20-default.list \
    https://mirrors.tuna.tsinghua.edu.cn/rosdistro/rosdep/sources.list.d/20-default.list

# 设置环境变量
export ROSDISTRO_INDEX_URL=https://mirrors.tuna.tsinghua.edu.cn/rosdistro/index-v4.yaml

# 更新
rosdep update
```

### 13.2 pip安装速度慢

**解决方法:**
```bash
# 使用清华镜像
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple <package>

# 或设置为默认
pip3 config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### 13.3 CAN模块加载失败

**问题:** `modprobe: FATAL: Module can not found`

**解决方法:**
```bash
# 检查内核版本
uname -r

# 安装内核模块
sudo apt install -y linux-modules-extra-$(uname -r)

# 重启后再次尝试
sudo reboot
```
