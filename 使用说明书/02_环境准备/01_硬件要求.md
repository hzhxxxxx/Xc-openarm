# 01_硬件要求

## OpenArmX 系统硬件要求

本文档详细说明运行 OpenArmX 双臂机器人系统所需的硬件配置和推荐设备。

---

## 1. 控制计算机要求

### 1.1 最低配置

| 组件 | 最低要求 | 说明 |
|------|---------|------|
| **CPU** | Intel Core i5 / AMD Ryzen 5 | 4核心, 2.0 GHz以上 |
| **内存** | 8 GB RAM | DDR4或更高 |
| **存储** | 50 GB 可用空间 | HDD可用,SSD推荐 |
| **操作系统** | Ubuntu 22.04 LTS | 64位(amd64) |
| **USB接口** | USB 2.0 × 2 | 用于CAN转换器 |
| **网络** | 以太网接口 | 可选,用于远程访问 |

**⚠️ 注意:** 最低配置可以运行系统,但可能存在以下限制:
- MoveIt规划速度较慢
- RViz可视化可能卡顿
- 不适合复杂的多任务场景

### 1.2 推荐配置

| 组件 | 推荐配置 | 说明 |
|------|---------|------|
| **CPU** | Intel Core i7 / AMD Ryzen 7 | 8核心, 3.0 GHz以上 |
| **内存** | 16 GB RAM 或更多 | 用于MoveIt和RViz |
| **存储** | 100 GB SSD | 提升系统响应速度 |
| **显卡** | 独立显卡(可选) | 改善RViz渲染性能 |
| **操作系统** | Ubuntu 22.04 + RT补丁 | 实时性能优化 |
| **USB接口** | USB 3.0 × 2+ | 更快的数据传输 |

**优势:**
- ✅ 流畅的可视化体验
- ✅ 快速的轨迹规划
- ✅ 支持多任务并行
- ✅ 更好的实时性能

### 1.3 专业配置

**用于复杂应用(视觉伺服、机器学习等):**

| 组件 | 专业配置 |
|------|---------|
| **CPU** | Intel Core i9 / AMD Ryzen 9 (12核心+) |
| **内存** | 32 GB RAM 或更多 |
| **显卡** | NVIDIA RTX系列(CUDA支持) |
| **存储** | 500 GB NVMe SSD |
| **操作系统** | Ubuntu 22.04 + RT补丁 + CUDA |

## 2. CAN通信设备

### 2.1 CAN-USB转换器

OpenArmX 需要**双通道CAN-USB转换器**来控制左右臂:

**推荐型号:**
1. **USBCAN-II 双通道**
   - 支持CAN 2.0
   - 波特率最高1 Mbps
   - Linux SocketCAN驱动支持

2. **Peak PCAN-USB FD**
   - 高性能
   - 良好的Linux支持
   - 支持CAN FD(未来扩展)

3. **Kvaser Leaf Light HS v2**
   - 工业级可靠性
   - 优秀的驱动支持

**必需规格:**
```
□ 通道数量: 至少2个独立CAN通道
□ 波特率: 支持1 Mbps
□ 接口: USB(推荐USB 3.0)
□ 驱动: Linux SocketCAN驱动支持
□ 隔离: 推荐带电气隔离
```

**⚠️ 重要提示:**
- 不推荐使用单通道转换器(需要两个设备)
- 确认Linux驱动兼容性
- 购买时确认支持SocketCAN

### 2.2 CAN线缆

**规格要求:**
```
□ 类型: 双绞线屏蔽线缆
□ 特性阻抗: 120Ω
□ 线径: ≥ 0.25 mm²
□ 长度: 尽量短(<5m)
□ 连接器: DB9或端子(根据设备)
```

**推荐配置:**
- CAN_H, CAN_L 双绞
- 铝箔+编织网双重屏蔽
- 两端配120Ω终端电阻

**布线建议:**
- ✅ 远离强电线缆(>30cm)
- ✅ 避免与电源线平行走线
- ✅ 最小化环路面积
- ✅ 两端良好接地

### 2.3 CAN接口接线

**典型接线图:**

```
CAN转换器(DB9)          OpenArmX电机
┌─────────┐           ┌─────────┐
│  DB9    │           │ CAN端子 │
│ ┌─────┐ │           │         │
│ │ 7 2 │ │  CAN_H   │  CAN_H  │
│ │6 1  │ │◄─────────┤         │
│ │ 8 3 │ │  CAN_L   │  CAN_L  │
│ │  9 4│ │◄─────────┤         │
│ │   5 │ │  GND     │  GND    │
│ └─────┘ │◄─────────┤         │
└─────────┘           └─────────┘

DB9针脚定义(常见):
Pin 2: CAN_L
Pin 3: GND
Pin 7: CAN_H
```

**终端电阻连接:**
```
总线起点                            总线末端
  120Ω                                120Ω
   │                                    │
CAN_H ─────────────────────────────── CAN_H
CAN_L ─────────────────────────────── CAN_L
```

## 3. 电源系统

### 3.1 主电源要求

**规格:**
```
输入电压: AC 100-240V (通用)
输出电压: DC 48V
输出电流: ≥ 20A (推荐 25A+)
输出功率: ≥ 1000W
```

**推荐型号:**
- 明纬(MEAN WELL) RSP-1500-48
- 台达(DELTA) PMT-48V20W2BA
- TDK-Lambda HWS1000-48

**安全特性:**
```
□ 过压保护 (OVP)
□ 过流保护 (OCP)
□ 过温保护 (OTP)
□ 短路保护 (SCP)
□ EMI滤波
```

### 3.2 电源容量计算

**最大功耗估算:**

```
单个RS04电机: ~100W (峰值)
单个RS03电机: ~60W (峰值)
单个RS00电机: ~30W (峰值)

总计:
RS04 × 4 (关节1-2) = 400W
RS03 × 4 (关节3-4) = 240W
RS00 × 8 (关节5-8) = 240W
─────────────
合计: 880W (峰值)

推荐电源: 880W × 1.5 = 1320W
```

**⚠️ 注意:**
- 上述为理论峰值,实际很少同时达到
- 典型运行功耗约400-600W
- 仍建议1000W+电源保证稳定性

### 3.3 配电建议

**推荐配电方案:**

```
AC 220V ──► 电源开关 ──► 48V电源 ──┬──► 机械臂电机
                                    │
                                    ├──► 急停继电器 ──► 电机使能
                                    │
                                    └──► 控制计算机(可选)
```

**安全措施:**
1. **急停按钮**
   - 红色蘑菇头按钮
   - 机械自锁式
   - 直接切断48V供电

2. **保险丝**
   - 主电源输入: 10A保险丝(AC端)
   - 电机供电: 25A断路器(DC端)

3. **接地**
   - 电源外壳必须接地
   - 机器人基座接地
   - 控制计算机接地

## 4. 安全设备

### 4.1 急停系统

**必需设备:**

**急停按钮:**
```
类型: 红色蘑菇头按钮
触点: 常闭触点(NC)
规格: ≥ 10A @ 48V DC
安装: 手臂伸展可及范围内
动作: 机械自锁,旋转解锁
```

**推荐型号:**
- 施耐德(Schneider) XB4BS542
- OMRON A22E-M
- ABB CE4T

**接线方式:**
```
48V+ ───► 急停按钮NC ───► 电机供电
                │
              按下时断开
```

### 4.2 防护设施

**推荐安全设施:**

1. **物理围栏**(推荐)
   - 高度: ≥ 1.2m
   - 材质: 铝型材+透明亚克力板
   - 开口: 带安全门锁

2. **警示标识**
   - 黄色警示标志
   - "机器人工作区域,请勿进入"
   - LED警示灯(可选)

3. **地面标线**
   - 黄黑警示地胶
   - 标注安全距离(≥1m)

4. **应急设备**
   - 灭火器(CO₂或干粉)
   - 急救箱
   - 应急照明

### 4.3 个人防护装备(PPE)

**推荐穿戴:**
```
□ 防护眼镜(必需)
□ 实验服/工作服(推荐)
□ 防滑鞋(推荐)
□ 绝缘手套(维护时必需)
```

**禁止穿戴:**
```
✘ 宽松衣物
✘ 长项链、手链
✘ 散披长发
```

### 5.2 环境要求

**温度:**
```
工作温度: 5°C ~ 40°C
最佳温度: 15°C ~ 30°C
存储温度: -10°C ~ 60°C
```

**湿度:**
```
工作湿度: 20% ~ 80% RH(无冷凝)
最佳湿度: 40% ~ 60% RH
```

**其他要求:**
```
□ 室内使用
□ 无腐蚀性气体
□ 无导电粉尘

□ 无剧烈振动

**推荐规格:**

尺寸: ≥ 1.5m × 1.0m

承重: ≥ 100kg
稳定性: 坚固,无晃动
表面: 平整,防滑
```

**安装固定:**
- 机器人基座用螺栓固定在台面上
- 台面固定在地面或墙壁上
- 防止运动时整体移动
