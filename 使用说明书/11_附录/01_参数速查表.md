# 01_参数速查表

## OpenArmX常用参数快速参考

---

## 1. 电机参数

### 1.1 电机型号参数

| 型号 | 功率 | 额定扭矩 | 峰值扭矩 | 最大速度 | 用途 |
|------|------|---------|---------|---------|------|
| RS04 | 700W | 40 Nm | 120 Nm | 15 rad/s | 肩部(1-2) |
| RS03 | 380W | 20 Nm | 60 Nm | 20 rad/s | 大臂(3-4) |
| RS00 | 140W | 5 Nm | 14 Nm | 33 rad/s | 小臂-夹爪(5-8) |

### 1.2 电机ID映射

**can0 (左臂):**
```
ID 1-2: 肩部 (RS04)
ID 3-4: 大臂 (RS03)
ID 5-8: 小臂-夹爪 (RS00)
```

**can1 (右臂):**
```
ID 1-2: 肩部 (RS04)
ID 3-4: 大臂 (RS03)
ID 5-8: 小臂-夹爪 (RS00)
```

## 2. 控制参数

### 2.1 推荐KP/KD参数

| 应用场景 | 关节1-4 KP | 关节1-4 KD | 关节5-7 KP | 关节5-7 KD | 夹爪 KP | 夹爪 KD |
|---------|-----------|-----------|-----------|-----------|---------|---------|
| 示教模式 | 1.0 | 0.1 | 0.1 | 0.05 | 0.1 | 0.05 |
| 常规运动(默认) | 50.0 | 2.5 | 10.0 | 0.5 | 5.0 | 0.5 |
| 高刚性模式 | 500.0 | 10.0 | 50.0 | 1.0 | 50.0 | 1.0 |

**说明:**
- 关节1-4: RS04/RS03电机（肩部和大臂）
- 关节5-7: RS00电机（小臂）
- 夹爪: RS00电机（关节8）
- 参数来源: `openarmx_hardware/include/openarmx_hardware/v10_simple_hardware.hpp`

### 2.2 速度和加速度限制

```yaml
# 推荐值
joint_limits:
  velocity: 2.0 rad/s        # 初次: 0.5
  acceleration: 1.0 rad/s²   # 初次: 0.3
```

### 2.3 MIT模式参数范围

```
位置: ±12.57 rad (所有型号)
速度: RS04: ±15 rad/s, RS03: ±20 rad/s, RS00: ±33 rad/s
KP:   RS04/RS03: 0-5000, RS00: 0-500
KD:   RS04/RS03: 0-100, RS00: 0-5
```

## 3. CAN总线参数

### 3.1 CAN接口配置

| 参数 | can0 | can1 |
|------|------|------|
| 波特率 | 1 Mbps | 1 Mbps |
| 设备数 | 8 | 8 |
| 终端电阻 | 60Ω | 60Ω |

### 3.2 CAN命令

```bash
# 启动
sudo ip link set can0 type can bitrate 1000000
sudo ip link set can0 up

# 关闭
sudo ip link set can0 down

# 状态
ip -s link show can0
```

## 5. 电源参数

### 5.1 电源规格

```
输入: AC 100-240V
输出: DC 48V
电流: ≥20A (推荐25A+)
功率: ≥1000W (推荐1500W)
```

### 5.2 功耗估算

```
单电机峰值:
- RS04: ~100W
- RS03: ~60W
- RS00: ~30W

系统总峰值:
- RS04 × 4 = 400W
- RS03 × 4 = 240W
- RS00 × 8 = 240W
- 总计: ~880W

典型运行: 400-600W
```

## 6. 温度限制

| 温度范围 | 状态 | 处理 |
|---------|------|------|
| <50°C | 🟢 正常 | 无 |

## 10. 错误码

### 10.1 电机错误码

| 错误码 | 含义 | 处理 |
|-------|------|------|
| 0x00 | 正常 | 无 |
| 0x01 | 过压 | 检查电源 |
| 0x02 | 欠压 | 检查电源 |
| 0x04 | 过流 | 降低负载 |
| 0x08 | 过温 | 冷却 |
| 0x10 | 编码器故障 | 检查/更换 |
| 0x20 | 通信超时 | 检查CAN |
