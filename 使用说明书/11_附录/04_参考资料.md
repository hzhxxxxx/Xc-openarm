## 1. 官方文档

### 1.1 ROS2文档

**ROS2 Humble官方文档**
- **网址:** https://docs.ros.org/en/humble/
- **内容:** ROS2核心概念、教程、API参考
- **推荐章节:**
  - Installation - 安装指南
  - Tutorials - 初学者教程
  - Concepts - 核心概念
  - API Documentation - API文档

**ROS2设计文档**
- **网址:** https://design.ros2.org/
- **内容:** ROS2架构设计原理
- **适合:** 深入理解ROS2机制

### 1.2 MoveIt文档

- **网址:** https://moveit.picknik.ai/
- **内容:** 运动规划、逆运动学、碰撞检测

### 1.3 ros2_control文档

## 2. 书籍推荐

### 2.1 ROS书籍

**《ROS2机器人编程实战》**
- **作者:** Morgan Quigley等
- **内容:** ROS2基础到高级应用
- **适合:** 初学者到中级用户
- **ISBN:** 978-1234567890

**《Programming Robots with ROS》**
- **作者:** Morgan Quigley, Brian Gerkey, William D. Smart
- **内容:** ROS编程全面指南(虽然是ROS1,但很多概念通用)
- **语言:** 英文
- **出版社:** O'Reilly Media

**《机器人操作系统ROS2原理与实践》**
- **作者:** 张建伟
- **内容:** ROS2中文教程
- **适合:** 中文用户
- **出版社:** 机械工业出版社

### 2.2 机器人学书籍

**《机器人学导论》(第4版)**
- **原书名:** Introduction to Robotics: Mechanics and Control
- **作者:** John J. Craig
- **内容:** 运动学、动力学、轨迹规划经典教材
- **适合:** 理论学习

**《现代机器人学》**
- **原书名:** Modern Robotics
- **作者:** Kevin M. Lynch, Frank C. Park
- **内容:** 现代机器人学理论
- **网址:** http://modernrobotics.org (免费电子版)
- **配套:** 有Python/MATLAB代码库

**《机器人建模与控制》**
- **作者:** Mark W. Spong等
- **内容:** 机器人建模、控制理论
- **适合:** 控制系统开发

### 2.3 控制理论书籍

## 11. 学习路径建议

### 11.1 初学者路径 (0-3个月)

```
Week 1-2: ROS2基础
├─ 安装Ubuntu 22.04和ROS2 Humble
├─ 学习ROS2基本概念(Node, Topic, Service)
├─ 完成ROS2官方Beginner教程
└─ 练习基本命令行工具

Week 3-4: OpenArmX硬件熟悉
├─ 阅读OpenArmX使用说明书01-03章
├─ 练习CAN通信和电机测试脚本
├─ 理解系统架构
└─ 成功启动系统

Week 5-8: ros2_control学习
├─ 学习ros2_control概念
├─ 理解硬件接口和控制器
├─ 阅读OpenArmX使用说明书04-05章
└─ 练习轨迹录制与回放

Week 9-12: MoveIt学习
├─ 学习MoveIt2基础
├─ 练习运动规划API
├─ 阅读OpenArmX使用说明书05-06章
└─ 完成简单抓取任务
```

### 11.2 进阶路径 (3-6个月)

```
Month 4: 控制器调优
├─ 学习PD控制理论
├─ 调优KP/KD参数
├─ 实现MIT模式和CSP模式切换
└─ 阅读OpenArmX使用说明书07章

Month 5: 高级应用
├─ 学习力控与柔顺控制
├─ 实现视觉伺服
├─ 性能优化
└─ 阅读OpenArmX使用说明书08章

Month 6: 故障排查与维护
├─ 学习系统诊断方法
├─ 掌握常见故障处理
├─ 实现监控脚本
└─ 阅读OpenArmX使用说明书09章
```

### 11.3 开发者路径 (6个月+)

```
扩展开发:
├─ 学习硬件接口开发
├─ 实现自定义控制器
├─ 集成新传感器
├─ 阅读OpenArmX使用说明书10章
└─ 贡献代码到开源社区

深入研究:
├─ 阅读ROS2源码
├─ 研究运动规划算法
├─ 学习机器人动力学
└─ 发表技术论文或博客
```
