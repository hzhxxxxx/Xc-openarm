# 06_回零操作

## 机器人回零操作详解

将机器人从任意姿态安全平滑地回到零位姿态。

---

## 1. 回零概述

### 1.1 什么是回零

**回零(Homing):** 将机器人的所有关节从当前位置运动到预定义的零位姿态。

**零位姿态定义:**
```
左臂/右臂:
  - 肩部关节(1-3): 0 rad (手臂垂直向下)
  - 肘部关节(4): 0 rad (肘部伸直)
  - 腕部关节(5-7): 0 rad (腕部中立)
  - 夹爪(8): 0 rad (打开状态)

物理描述:
双臂自然垂直下垂,腕部保持中立位置,夹爪打开
```

### 1.2 回零的作用

```
✓ 系统启动后的初始化
✓ 测试前的标准起始姿态
✓ 异常后的安全恢复
✓ 验证零位校准是否准确
✓ 日常操作的参考姿态
```

### 1.3 回零前提条件

```
□ CAN接口已启动
□ 电机已使能
□ 零位已正确校准
□ 周围无障碍物
□ 准备急停措施
```

## 3. 分步回零逻辑

### 3.1 为什么分阶段

**原因:**
```
1. 避免关节间干涉
   - 先回肩部避免碰撞
   - 再回肘部避免奇异位形
   - 最后回夹爪

2. 降低CAN总线负载
   - 分批发送命令
   - 避免消息拥塞

3. 便于监控和调试
   - 逐步验证
   - 发现问题易定位
```

### 3.2 回零顺序

**标准顺序:**
```
阶段1: 肩部3关节 (ID 1-3)
       ↓
阶段2: 肘部1关节 + 腕部3关节 (ID 4-7)
       ↓
阶段3: 夹爪 (ID 8)
```

**示意图:**
```
初始姿态(任意)
    ↓
[阶段1] 肩部回正,手臂大致垂直
    ↓
[阶段2] 肘部伸直,腕部归中
    ↓
[阶段3] 夹爪打开
    ↓
零位姿态(标准)
```

### 3.3 平滑运动

**脚本实现平滑性的方法:**
```python
# 1. 使用适中的KP/KD(不过大)
kp = 20.0  # 不是100,避免震荡

# 2. MIT模式自然加减速
# 电机内部会平滑加减速

# 3. 分阶段等待
time.sleep(stage_duration)
# 确保上一阶段完成再进行下一阶段
```

## 4. 手动实现回零

### 4.1 单臂手动回零

**如果脚本不可用,手动实现:**

```python
#!/usr/bin/env python3
"""
手动回零脚本示例
"""
import can
from robstride import Motor
import time

# 初始化CAN总线
bus0 = can.interface.Bus(channel='can0', bustype='socketcan')

# 创建左臂8个电机对象
left_motors = [Motor(bus0, i) for i in range(1, 9)]

# 确认电机已使能
print("确保电机已使能(en_all_motors.py)")
input("按回车继续...")

# 读取当前位置
print("\n当前位置:")
for i, motor in enumerate(left_motors):
    fb = motor.get_feedback()
    print(f"Motor {i+1}: {fb['position']:.3f} rad")

input("\n按回车开始回零...")

# 阶段1: 肩部回零
print("\n[阶段1] 肩部回零...")
kp, kd = 20.0, 1.0
for motor in left_motors[:3]:
    motor.set_position(0.0, vel=0.0, kp=kp, kd=kd, torque=0.0)
    time.sleep(0.1)

time.sleep(5)

# 阶段2: 肘腕回零
print("[阶段2] 肘腕回零...")
for motor in left_motors[3:7]:
    motor.set_position(0.0, vel=0.0, kp=kp, kd=kd, torque=0.0)
    time.sleep(0.1)

time.sleep(5)

# 阶段3: 夹爪回零
print("[阶段3] 夹爪回零...")
kp_gripper, kd_gripper = 10.0, 0.5
left_motors[7].set_position(0.0, vel=0.0, kp=kp_gripper, kd=kd_gripper, torque=0.0)

time.sleep(3)

# 验证
print("\n回零后位置:")
max_error = 0.0
for i, motor in enumerate(left_motors):
    fb = motor.get_feedback()
    error = abs(fb['position'] - 0.0)
    max_error = max(max_error, error)
    print(f"Motor {i+1}: {fb['position']:.4f} rad (误差: {error:.4f})")

print(f"\n最大误差: {max_error:.4f} rad")

if max_error < 0.05:
    print("✓ 回零成功!")
else:
    print("✗ 误差较大,可能需要重新校准零位")

print("\n回零完成")
```

### 4.2 双臂同步回零

```python
#!/usr/bin/env python3
"""
双臂同步回零
"""
import can
from robstride import Motor
import time

# 双CAN总线
bus0 = can.interface.Bus(channel='can0', bustype='socketcan')
bus1 = can.interface.Bus(channel='can1', bustype='socketcan')

# 左右臂
left_motors = [Motor(bus0, i) for i in range(1, 9)]
right_motors = [Motor(bus1, i) for i in range(1, 9)]

print("OpenArmX 双臂同步回零\n")

# 阶段1: 双臂肩部
print("[阶段1] 双臂肩部回零...")
kp, kd = 20.0, 1.0

for motor in left_motors[:3] + right_motors[:3]:
    motor.set_position(0.0, vel=0.0, kp=kp, kd=kd, torque=0.0)
    time.sleep(0.05)

time.sleep(5)

# 阶段2: 双臂肘腕
print("[阶段2] 双臂肘腕回零...")
for motor in left_motors[3:7] + right_motors[3:7]:
    motor.set_position(0.0, vel=0.0, kp=kp, kd=kd, torque=0.0)
    time.sleep(0.05)

time.sleep(5)

# 阶段3: 双夹爪
print("[阶段3] 双夹爪回零...")
kp_g, kd_g = 10.0, 0.5
left_motors[7].set_position(0.0, vel=0.0, kp=kp_g, kd=kd_g, torque=0.0)
right_motors[7].set_position(0.0, vel=0.0, kp=kp_g, kd=kd_g, torque=0.0)

time.sleep(3)

# 验证双臂同步性
print("\n双臂位置验证:")
for i in range(8):
    left_fb = left_motors[i].get_feedback()
    right_fb = right_motors[i].get_feedback()

    left_pos = left_fb['position']
    right_pos = right_fb['position']

    sync_error = abs(left_pos - right_pos)

    print(f"Motor {i+1}: L={left_pos:.4f}, R={right_pos:.4f}, 差异={sync_error:.4f} rad")

print("\n双臂回零完成")
```

### 5.2 超时保护

```python
import time

def safe_homing_with_timeout(motor, timeout=10.0):
    """
    带超时的回零
    """
    start_time = time.time()

    motor.set_position(0.0, vel=0.0, kp=20.0, kd=1.0, torque=0.0)

    while (time.time() - start_time) < timeout:
        fb = motor.get_feedback()
        error = abs(fb['position'] - 0.0)

        if error < 0.05:  # 到达目标
            print(f"Motor {motor.motor_id} 回零完成")
            return True

        time.sleep(0.1)

    # 超时
    print(f"警告: Motor {motor.motor_id} 回零超时!")
    return False
```

### 6.3 重复性测试

```python
# 测试回零重复性
import numpy as np

def test_homing_repeatability(motors, n_trials=5):
    """
    测试多次回零的重复性
    """
    positions_history = []

    for trial in range(n_trials):
        print(f"\n第{trial+1}次回零...")

        # 先移动到随机位置
        for motor in motors:
            random_pos = np.random.uniform(-0.5, 0.5)
            motor.set_position(random_pos, vel=0.0, kp=20.0, kd=1.0, torque=0.0)
            time.sleep(0.1)

        time.sleep(3)

        # 执行回零
        for motor in motors:
            motor.set_position(0.0, vel=0.0, kp=20.0, kd=1.0, torque=0.0)
            time.sleep(0.1)

        time.sleep(5)

        # 记录位置
        positions = [motor.get_feedback()['position'] for motor in motors]
        positions_history.append(positions)

        print(f"位置: {[f'{p:.4f}' for p in positions]}")

    # 分析重复性
    positions_history = np.array(positions_history)  # (n_trials, n_motors)

    print("\n重复性分析:")
    for i in range(len(motors)):
        std = np.std(positions_history[:, i])
        print(f"Motor {i+1}: 标准差={std:.5f} rad")

    overall_std = np.mean(np.std(positions_history, axis=0))
    print(f"\n总体重复性: {overall_std:.5f} rad")

    if overall_std < 0.01:
        print("✓ 重复性优秀")
    elif overall_std < 0.05:
        print("✓ 重复性良好")
    else:
        print("✗ 重复性较差,需检查")

# 运行测试
test_homing_repeatability(left_motors[:3], n_trials=5)
```

**原因:**
- 零位校准不准确
- 机械装配松动
- 编码器漂移

**解决:**
