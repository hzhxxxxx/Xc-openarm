## 2. 状态参数解读

### 2.1 位置(Position)

**单位:** rad(弧度)

**正常范围:**
- 静止时应该稳定(变化<0.01 rad)
- 运动时连续变化
- 零位姿态接近0

**异常情况:**
- 静止时大幅跳变 → 编码器故障
- 数值明显偏离预期 → 零位丢失

### 2.2 速度(Velocity)

**单位:** rad/s

**正常状态:**
- 静止时应为0或很小(<0.1 rad/s)
- 运动时为设定值

**异常情况:**
- 静止时速度大 → 电机抖动或失控

### 2.3 力矩(Torque)

**单位:** Nm(牛·米)

**正常范围:**
- 空载: 0~2 Nm
- 负载: 根据重量增加
- 峰值: 不超过额定扭矩

**异常情况:**
- 静止时扭矩过大 → KP设置过高或卡顿
- 持续峰值 → 过载

### 2.4 温度(Temperature)

**单位:** °C

**安全范围:**
| 温度 | 状态 | 处理 |
|------|------|------|
| <50°C | 🟢 正常 | 无需处理 |
| 50-60°C | 🟡 温和 | 注意监控 |
| 60-70°C | 🟡 偏高 | 降低负载 |
| 70-80°C | 🟠 警告 | 停止高强度运动 |
| >80°C | 🔴 危险 | 立即停止并冷却 |

**影响因素:**
- 环境温度
- 运动强度
- KP/KD参数
- 散热条件

### 2.5 错误码(Error Code)

**常见错误码:**
| 错误码 | 含义 | 处理 |
|-------|------|------|
| 0x00 | 正常 | 无 |
| 0x01 | 过压 | 检查电源 |
| 0x02 | 欠压 | 检查电源 |
| 0x04 | 过流 | 降低负载/KP |
| 0x08 | 过温 | 冷却 |
| 0x10 | 编码器故障 | 检查/更换 |
| 0x20 | CAN超时 | 检查通信 |

## 3. 持续监控

### 3.1 实时监控脚本

```bash
# 创建实时监控
cat > ~/monitor_motors.sh << 'EOF'
#!/bin/bash
cd ~/openarmx_robotstride_ws-cc/src/motor_tests_openarmx_com

while true; do
    clear
    echo "===== OpenArmX 电机监控 ====="
    date
    echo ""
    python3 check_motor_status.py
    sleep 2
done
EOF

chmod +x ~/monitor_motors.sh

# 运行
./monitor_motors.sh
```

### 3.2 温度监控

```bash
# 仅监控温度
python3 check_motor_status.py | grep -E "ID|温度"
```

### 3.3 记录到日志

```bash
# 每分钟记录一次状态
while true; do
    echo "$(date)" >> motor_log.txt
    python3 check_motor_status.py >> motor_log.txt
    echo "" >> motor_log.txt
    sleep 60
done &
```

## 4. 异常诊断

### 4.1 温度异常

**高温处理:**
```bash
# 1. 停止运动
python3 dis_all_motors.py

# 2. 等待冷却(<50°C)

# 3. 检查原因:
# - 负载过大
# - KP/KD过高
# - 散热不良
# - 电机故障

# 4. 调整后重新测试
```

### 4.2 位置异常

**零位丢失:**
```bash
# 1. 回零测试
python3 control_motor_gohome.py

# 2. 如果姿态不正确,重新设置零位
python3 dis_all_motors.py
# 手动调整到零位
python3 en_all_motors.py
python3 set_motor_zeros_setMIT.py
```

```bash

python3 check_motor_status.py

# 2. 监听CAN总线
candump can0

# 3. 检查物理连接

# 4. 重启CAN和电机
```

## 5. 总结

**关键检查项:**
```
□ 所有电机都能读取状态
□ 温度在安全范围(<70°C)
□ 错误码为0x00
□ 位置反馈稳定
□ 速度和力矩合理
```

**监控频率:**
```
□ 每次启动后检查一次
□ 长时间运行每15分钟检查
□ 出现异常立即检查
```
