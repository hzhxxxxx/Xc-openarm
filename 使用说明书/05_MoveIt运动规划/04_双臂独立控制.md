## 1. 双臂控制概述

### 1.1 控制模式

OpenArmX支持以下双臂控制模式:

**独立控制 (Independent Control):**
```
左臂和右臂分别规划和执行
每个臂有独立的Planning Group
适用: 独立任务、测试、简单协作
```

**协调控制 (Coordinated Control):**
```
两臂同步规划和执行
需要编程实现时间同步
适用: 抓持大物体、双臂装配
```

**镜像控制 (Mirrored Control):**
```
右臂镜像左臂的运动
常用于对称操作
适用: 对称装配、双侧作业
```

### 1.2 Planning Groups

```
left_arm:
- 关节: left_joint_1 ~ left_joint_7
- 末端: left_link_7
- CAN: can0

right_arm:
- 关节: right_joint_1 ~ right_joint_7
- 末端: right_link_7
- CAN: can1

left_gripper:
- 关节: left_gripper_joint

right_gripper:
- 关节: right_gripper_joint
```

## 2. RViz双臂独立控制

### 2.1 分别规划左右臂

**步骤流程:**

```
阶段1: 规划左臂
├─ 选择Planning Group: left_arm
├─ 拖动交互标记设置左臂目标
├─ 点击 [Plan]
├─ 检查轨迹
└─ 点击 [Execute]

阶段2: 规划右臂
├─ 选择Planning Group: right_arm
├─ 拖动交互标记设置右臂目标
├─ 点击 [Plan]
├─ 检查轨迹
└─ 点击 [Execute]
```

**注意事项:**
```
⚠️ 左臂运动时,右臂保持当前位置
⚠️ 右臂运动时,左臂保持当前位置
⚠️ 需要手动确保两臂不会碰撞
⚠️ 执行顺序影响最终姿态
```

### 2.2 示例: 对称摆放

**任务:** 左右臂同时移动到对称位置

   X=0.4, Y=-0.3, Z=0.5

4. 选择Planning Group: right_arm

5. 拖动右臂到镜像位置
   X=0.4, Y=+0.3, Z=0.5

左右臂对称分布在基座两侧

## 3. Python双臂协调编程

### 3.1 基本双臂控制类

```python
#!/usr/bin/env python3
"""
双臂协调控制
"""
import rclpy
from rclpy.node import Node
from moveit_py import MoveGroupInterface
import geometry_msgs.msg
import threading

class DualArmController(Node):
    def __init__(self):
        super().__init__('dual_arm_controller')

        # 初始化MoveGroup
        self.left_arm = MoveGroupInterface(self, "left_arm")
        self.right_arm = MoveGroupInterface(self, "right_arm")
        self.left_gripper = MoveGroupInterface(self, "left_gripper")
        self.right_gripper = MoveGroupInterface(self, "right_gripper")

        # 设置速度
        for arm in [self.left_arm, self.right_arm]:
            arm.set_max_velocity_scaling_factor(0.3)
            arm.set_max_acceleration_scaling_factor(0.3)

        self.get_logger().info("双臂控制器初始化完成")

    def move_left_arm(self, x, y, z, roll, pitch, yaw):
        """移动左臂"""
        pose = self.create_pose(x, y, z, roll, pitch, yaw)
        self.left_arm.set_pose_target(pose)
        success = self.left_arm.go(wait=True)
        self.left_arm.clear_pose_targets()
        return success

    def move_right_arm(self, x, y, z, roll, pitch, yaw):
        """移动右臂"""
        pose = self.create_pose(x, y, z, roll, pitch, yaw)
        self.right_arm.set_pose_target(pose)
        success = self.right_arm.go(wait=True)
        self.right_arm.clear_pose_targets()
        return success

    def create_pose(self, x, y, z, roll, pitch, yaw):
        """创建位姿"""
        pose = geometry_msgs.msg.Pose()
        pose.position.x = x
        pose.position.y = y
        pose.position.z = z

        from tf_transformations import quaternion_from_euler
        q = quaternion_from_euler(roll, pitch, yaw)
        pose.orientation.x = q[0]
        pose.orientation.y = q[1]
        pose.orientation.z = q[2]
        pose.orientation.w = q[3]

        return pose

    def open_left_gripper(self):
        """打开左夹爪"""
        self.left_gripper.set_joint_value_target([0.0])
        return self.left_gripper.go(wait=True)

    def close_left_gripper(self):
        """闭合左夹爪"""
        self.left_gripper.set_joint_value_target([0.8])
        return self.left_gripper.go(wait=True)

    def open_right_gripper(self):
        """打开右夹爪"""
        self.right_gripper.set_joint_value_target([0.0])
        return self.right_gripper.go(wait=True)

    def close_right_gripper(self):
        """闭合右夹爪"""
        self.right_gripper.set_joint_value_target([0.8])
        return self.right_gripper.go(wait=True)

    def move_both_arms_sequential(self, left_pose, right_pose):
        """
        顺序移动双臂
        先左后右
        """
        self.get_logger().info("顺序移动: 左臂 → 右臂")

        # 左臂
        self.left_arm.set_pose_target(left_pose)
        left_success = self.left_arm.go(wait=True)
        self.left_arm.clear_pose_targets()

        if not left_success:
            self.get_logger().error("左臂移动失败")
            return False

        # 右臂
        self.right_arm.set_pose_target(right_pose)
        right_success = self.right_arm.go(wait=True)
        self.right_arm.clear_pose_targets()

        if not right_success:
            self.get_logger().error("右臂移动失败")
            return False

        self.get_logger().info("✓ 双臂顺序移动完成")
        return True

    def move_both_arms_parallel(self, left_pose, right_pose):
        """
        并行移动双臂
        同时规划和执行
        """
        self.get_logger().info("并行移动双臂")

        # 使用线程并行执行
        left_success = [False]
        right_success = [False]

        def move_left():
            self.left_arm.set_pose_target(left_pose)
            left_success[0] = self.left_arm.go(wait=True)
            self.left_arm.clear_pose_targets()

        def move_right():
            self.right_arm.set_pose_target(right_pose)
            right_success[0] = self.right_arm.go(wait=True)
            self.right_arm.clear_pose_targets()

        # 创建线程
        left_thread = threading.Thread(target=move_left)
        right_thread = threading.Thread(target=move_right)

        # 启动线程
        left_thread.start()
        right_thread.start()

        # 等待完成
        left_thread.join()
        right_thread.join()

        if left_success[0] and right_success[0]:
            self.get_logger().info("✓ 双臂并行移动完成")
            return True
        else:
            self.get_logger().error("双臂并行移动部分失败")
            return False

def main():
    rclpy.init()

    controller = DualArmController()

    import math

    # 示例1: 顺序控制
    left_pose = controller.create_pose(0.5, -0.3, 0.5, 0, math.pi/2, 0)
    right_pose = controller.create_pose(0.5, 0.3, 0.5, 0, math.pi/2, 0)

    controller.move_both_arms_sequential(left_pose, right_pose)

    # 示例2: 并行控制
    # controller.move_both_arms_parallel(left_pose, right_pose)

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 3.2 双臂协同抓取

**任务:** 双臂协作抓持大物体

```python
#!/usr/bin/env python3
"""
双臂协同抓取大物体
"""
import rclpy
from rclpy.node import Node
from moveit_py import MoveGroupInterface
import geometry_msgs.msg
import math
import time

class DualArmGrasp(Node):
    def __init__(self):
        super().__init__('dual_arm_grasp')

        self.left_arm = MoveGroupInterface(self, "left_arm")
        self.right_arm = MoveGroupInterface(self, "right_arm")
        self.left_gripper = MoveGroupInterface(self, "left_gripper")
        self.right_gripper = MoveGroupInterface(self, "right_gripper")

        for arm in [self.left_arm, self.right_arm]:
            arm.set_max_velocity_scaling_factor(0.2)
            arm.set_max_acceleration_scaling_factor(0.2)

        self.get_logger().info("双臂抓取控制器初始化")

    def create_pose(self, x, y, z, roll, pitch, yaw):
        pose = geometry_msgs.msg.Pose()
        pose.position.x = x
        pose.position.y = y
        pose.position.z = z

        from tf_transformations import quaternion_from_euler
        q = quaternion_from_euler(roll, pitch, yaw)
        pose.orientation.x = q[0]
        pose.orientation.y = q[1]
        pose.orientation.z = q[2]
        pose.orientation.w = q[3]

        return pose

    def grasp_large_object(self, object_x, object_y, object_z, object_width):
        """
        抓取大物体
        object_x, object_y, object_z: 物体中心位置
        object_width: 物体宽度(Y方向)
        """
        self.get_logger().info(f"开始抓取大物体: 中心({object_x}, {object_y}, {object_z}), 宽度={object_width}")

        # 计算左右抓取点
        left_grasp_y = object_y - object_width / 2 - 0.05   # 左侧
        right_grasp_y = object_y + object_width / 2 + 0.05  # 右侧

        # 阶段1: 移动到物体两侧上方
        self.get_logger().info("[1/6] 移动到物体两侧上方")

        left_above = self.create_pose(
            object_x, left_grasp_y, object_z + 0.15,
            0, math.pi/2, 0  # 夹爪朝下
        )

        right_above = self.create_pose(
            object_x, right_grasp_y, object_z + 0.15,
            0, math.pi/2, 0
        )

        # 顺序移动(避免碰撞)
        self.left_arm.set_pose_target(left_above)
        if not self.left_arm.go(wait=True):
            self.get_logger().error("左臂移动失败")
            return False
        self.left_arm.clear_pose_targets()

        self.right_arm.set_pose_target(right_above)
        if not self.right_arm.go(wait=True):
            self.get_logger().error("右臂移动失败")
            return False
        self.right_arm.clear_pose_targets()

        # 阶段2: 打开双夹爪
        self.get_logger().info("[2/6] 打开双夹爪")
        self.left_gripper.set_joint_value_target([0.0])
        self.left_gripper.go(wait=True)
        self.right_gripper.set_joint_value_target([0.0])
        self.right_gripper.go(wait=True)

        # 阶段3: 下降到抓取高度
        self.get_logger().info("[3/6] 下降到抓取高度")

        import copy
        left_waypoints = []
        left_waypoints.append(self.left_arm.get_current_pose().pose)
        left_grasp = copy.deepcopy(left_waypoints[0])
        left_grasp.position.z = object_z
        left_waypoints.append(left_grasp)

        right_waypoints = []
        right_waypoints.append(self.right_arm.get_current_pose().pose)
        right_grasp = copy.deepcopy(right_waypoints[0])
        right_grasp.position.z = object_z
        right_waypoints.append(right_grasp)

        # 笛卡尔路径
        (left_plan, left_frac) = self.left_arm.compute_cartesian_path(
            left_waypoints, 0.01, 0.0
        )
        (right_plan, right_frac) = self.right_arm.compute_cartesian_path(
            right_waypoints, 0.01, 0.0
        )

        if left_frac < 0.9 or right_frac < 0.9:
            self.get_logger().error("笛卡尔路径规划失败")
            return False

        # 同步下降
        self.left_arm.execute(left_plan, wait=False)   # 非阻塞
        self.right_arm.execute(right_plan, wait=True)  # 阻塞等待

        # 阶段4: 闭合双夹爪
        self.get_logger().info("[4/6] 闭合双夹爪抓取")
        self.left_gripper.set_joint_value_target([0.8])
        self.left_gripper.go(wait=True)
        self.right_gripper.set_joint_value_target([0.8])
        self.right_gripper.go(wait=True)

        time.sleep(1.0)  # 确保抓稳

        # 阶段5: 同步提升
        self.get_logger().info("[5/6] 同步提升物体")

        left_up_waypoints = []
        left_up_waypoints.append(self.left_arm.get_current_pose().pose)
        left_up = copy.deepcopy(left_up_waypoints[0])
        left_up.position.z += 0.2
        left_up_waypoints.append(left_up)

        right_up_waypoints = []
        right_up_waypoints.append(self.right_arm.get_current_pose().pose)
        right_up = copy.deepcopy(right_up_waypoints[0])
        right_up.position.z += 0.2
        right_up_waypoints.append(right_up)

        (left_plan, _) = self.left_arm.compute_cartesian_path(
            left_up_waypoints, 0.01, 0.0
        )
        (right_plan, _) = self.right_arm.compute_cartesian_path(
            right_up_waypoints, 0.01, 0.0
        )

        # 同步提升
        self.left_arm.execute(left_plan, wait=False)
        self.right_arm.execute(right_plan, wait=True)

        # 阶段6: 移动到目标位置(可选)
        self.get_logger().info("[6/6] 保持抓取姿态")
        time.sleep(2.0)

        self.get_logger().info("✓ 双臂协同抓取完成")
        return True

    def release_object(self):
        """释放物体"""
        self.get_logger().info("释放物体")

        # 打开双夹爪
        self.left_gripper.set_joint_value_target([0.0])
        self.left_gripper.go(wait=True)
        self.right_gripper.set_joint_value_target([0.0])
        self.right_gripper.go(wait=True)

        self.get_logger().info("✓ 物体已释放")

def main():
    rclpy.init()

    grasp_controller = DualArmGrasp()

    # 抓取物体
    # 物体: 中心(0.5, 0.0, 0.45), 宽度0.3m
    success = grasp_controller.grasp_large_object(
        object_x=0.5,
        object_y=0.0,
        object_z=0.45,
        object_width=0.3
    )

    if success:
        time.sleep(3)
        grasp_controller.release_object()

    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 3.3 镜像运动

```python
def mirror_motion(self, left_target_joints):
    """
    右臂镜像左臂运动
    left_target_joints: [j1, j2, j3, j4, j5, j6, j7]
    """
    # 镜像规则:
    # joint_1: 取反(肩部旋转)
    # 其他关节: 相同

    right_target_joints = [
        -left_target_joints[0],  # joint_1反向
        left_target_joints[1],
        left_target_joints[2],
        left_target_joints[3],
        left_target_joints[4],
        left_target_joints[5],
        left_target_joints[6]
    ]

    # 同时设置目标
    self.left_arm.set_joint_value_target(left_target_joints)
    self.right_arm.set_joint_value_target(right_target_joints)

    # 并行执行
    import threading

    def move_left():
        self.left_arm.go(wait=True)

    def move_right():
        self.right_arm.go(wait=True)

    left_thread = threading.Thread(target=move_left)
    right_thread = threading.Thread(target=move_right)

    left_thread.start()
    right_thread.start()

    left_thread.join()
    right_thread.join()

    self.get_logger().info("镜像运动完成")

### 4.2 运动时避碰

**策略:**

```
1. 顺序运动(推荐)
   - 先移动一臂
   - 再移动另一臂
   - 保守但安全

2. 分区工作
   - 左臂负责左侧工作空间
   - 右臂负责右侧工作空间
   - 避免交叉

3. 高度分离
   - 一臂在高处作业
   - 另一臂在低处作业
   - 垂直方向分离

4. 添加虚拟障碍物
   - 在两臂间添加虚拟墙
   - 强制MoveIt避让
```

def check_sync_error(self):
    """检查双臂同步误差"""
    left_pose = self.left_arm.get_current_pose().pose
    right_pose = self.right_arm.get_current_pose().pose

    # 假设对称任务,检查Y坐标
    sync_error_y = abs(left_pose.position.y + right_pose.position.y)

    self.get_logger().info(f"Y方向同步误差: {sync_error_y:.4f} m")

    if sync_error_y > 0.05:  # 5cm
        self.get_logger().warn("同步误差较大!")
        return False
    return True

    self.open_left_gripper()

    # 右臂移走
    self.move_right_arm(0.5, 0.3, 0.5, 0, math.pi/2, 0)

    self.get_logger().info("物体传递完成")
```

## 7. 常见问题

### 7.1 双臂运动不同步

**症状:** 并行执行时,两臂到达时间差异大

**解决:**
```python
# 使用相同的速度缩放
left_arm.set_max_velocity_scaling_factor(0.3)
right_arm.set_max_velocity_scaling_factor(0.3)

# 选择相似复杂度的路径
# 或分步执行,每步同步
```

### 7.2 碰撞检测过于敏感

**症状:** MoveIt频繁报告碰撞,阻止合理运动

**解决:**
```python
# 调整碰撞检测padding
# 编辑 openarm_v10.srdf
<disable_collisions link1="left_link_4" link2="right_link_4" reason="Never"/>

# 或在代码中临时调整
scene.allow_collision("left_link_4", "right_link_4")
```

### 7.3 规划失败率高

**双臂同时规划时失败**

**原因:** 组合搜索空间大(14自由度)

**解决:**
```
1. 分步规划(先左后右)
2. 使用关节空间目标
3. 增加规划时间
4. 添加中间路径点
```
